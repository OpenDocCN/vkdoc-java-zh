# 3. 在本地执行项目

现在你已经启动并运行了依赖项，是时候启动聊天应用程序了。为此，你只需要下载 jar 文件并执行它。

![A453568_1_En_3_Figa_HTML.jpg](img/A453568_1_En_3_Figa_HTML.jpg) 确保你的机器上至少安装了 JDK 8。

```
$ wget https://github.com/jorgeacetozi/ebook-chat-app-spring-websocket-cassandra-redis/releases/download/ebook-chat-1.0.0/ebook-chat-1.0.0.jar && java -jar ebook-chat-1.0.0.jar
```

就是这样。打开你的浏览器并访问 `http://localhost:8080`。恭喜！现在你可以开始聊天了。

![A453568_1_En_3_Figb_HTML.jpg](img/A453568_1_En_3_Figb_HTML.jpg) 该聊天应用程序是使用 Google Chrome¹ 创建和测试的，因此我建议你使用 Chrome 运行该应用程序。

![A453568_1_En_3_Figc_HTML.jpg](img/A453568_1_En_3_Figc_HTML.jpg) 在学习了如何设置开发环境之后，你将能够使用 Apache Maven 从源代码创建 jar 文件。我提供这个 `ebook-chat-1.0.0.jar` 版本是为了方便你进行设置。
