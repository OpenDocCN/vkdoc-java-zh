# 10. 表示层/视图

用户通过他们的网页浏览器与您的应用程序进行交互。网页浏览器的作用是显示 HTML、JavaScript 和图像。每个网页文档都包含我们称之为头部（head）和主体（body）的两个部分。

头部包含标题标签（title tag）、链接标签（link tag）和脚本标签（script tag）。链接标签和脚本标签分别允许页面包含用于样式表和 JavaScript 的外部文件。我们的应用程序将包含指向 Bootstrap 样式表和 JavaScript 的链接。

文档的主体包含更多信息。主体是浏览器中可见内容的存放位置。这包括表单、显示给用户的文本、链接和图像。主体中的所有内容都将使用标签进行标记，这些标签决定了内容将如何渲染。

您可以在网页中包含注释，这些注释不会显示给用户。它们有助于您记录页面布局。
