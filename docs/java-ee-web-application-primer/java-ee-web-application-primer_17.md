# 17. 层叠样式表 (CSS)

层叠样式表 (CSS) 允许你指定 Web 应用程序的视觉样式和呈现方式。CSS 允许你将样式与结构分离。这意味着在处理页面时，你需要查看的代码更少。样式与结构和内容的分离也提高了可维护性。层叠样式表是一组可编程规则，用于定义网页如何显示内容。CSS 描述的样式包括颜色、字体、布局以及文档的其他呈现方面，包括针对不同设备和屏幕尺寸的显示变化。一个 CSS 文件可以描述适用于多个文档的通用样式。

通常，HTML 文件中的一个元素会有一个可以应用于它的 CSS 样式规则“层叠”。样式根据定义的位置进行层叠。如果你在多个位置定义了样式，则应用最后一个定义。你可以将 CSS 放在文档的 `<head>...</head>` 标签之间、外部样式表中（在你的服务器或其他服务器上），或者作为页面元素的样式属性。

通常建议使用外部样式表。要将外部样式表链接到你的文档，请在文档的 `<head>...</head>` 标签之间添加一个指向样式表的链接。将样式定义与 HTML 内容分开可以减少重复，并使你的网站更易于维护。

创建样式表时，你需要按名称、类或 ID 为每个元素创建规则。这些值被设置为元素的属性。浏览器在渲染页面时会应用 CSS 规则。每个规则有两个部分：一个选择器和一组用大括号括起来的一个或多个声明。每个声明由一个属性名和值对组成。一个规则中可以有多个声明。参见清单 17-1，其中展示了一个示例 CSS 规则，该规则将应用于文档中的所有 `span` 标签。此规则可以放在网页的 `<head>` 部分的 `<style>...</style>` 标签之间，或者放在一个单独的文件中。

```
span {
font-weight: bold;
color: yellow;
background-color: black;
}
```

## Span 与 Div 标签

`Span` 和 `div` 是用于定义文档各部分内容的容器标签。使用 `span` 和 `div` 可以为 JSP 或 HTML 页面的某个区域应用样式。将页面划分为页眉、正文和页脚等部分后，页面会更有条理。

`<div>` 标签用于将 HTML 页面划分为多个区块，因此可以封装各种元素。`<span>` 标签用于对文档中的内联元素进行分组。`<div>` 和 `<span>` 标签本身不会带来任何视觉变化。当每个标签包含 `style`、`class` 或 `id` 属性时，这些标签提供了一种控制文档部分内容样式的方式。ID 和 class 的区别在于，ID 用于标识单个元素，而 class 可用于标识多个元素。当你希望对多个元素应用同一样式时，请将样式指定为 class，因为只能有一个元素拥有特定值的 ID 属性，但多个元素可以在其 class 属性中共享相同的值。`<span>` 和 `<div>` 标签没有必需的属性。最常用的属性如下：

*   `style` 指定一个样式，该样式应用于直到对应 `结束` 标签为止的所有内容和元素。
*   `class` 指定一个 CSS 类，该类应用于直到 `结束` 标签为止的所有内容和元素。class 属性的值是在样式表文件中指定的 CSS 类。在样式表中，类名前面有一个点号。
*   `id` 用于标识标签，以便你可以使用 jQuery 或 JavaScript 选择它。任何元素的 `id` 属性都必须是唯一的。在样式表中，ID 名称前面有一个 `#` 井号。
