# 11. 使用 HTML 设计网页

HTML（超文本标记语言）是一种用于创建网页文档（网页）的标记语言。浏览器的主要功能是从网络服务器接收 HTML 格式的网页并进行显示。浏览器会应用由 HTML 标签和 CSS 属性值指定的所有字体、样式和布局。这些内容可以通过使用 JavaScript（浏览器的编程语言）进行进一步操作。

文档对象模型（DOM）是您 HTML 文档的一种树形结构表示。DOM 将每个节点视为代表文档一部分的对象。这些对象可以通过 JavaScript 以编程方式进行操作，从而使您的页面能够与用户交互。

以下是您需要了解的关于 HTML 的全部内容：

*   HTML 文档由称为标签的元素组成。
*   网页文档中 HTML 元素的集合称为 DOM（文档对象模型）。
*   标签用于标识文档内容和结构。
*   标签通常包含为元素提供参数的属性。
*   包含内容和标签的 HTML 文档由浏览器渲染，以显示网页的格式化版本。
*   HTML 的最新版本称为 HTML5。
