# 3. 安装和运行 Eclipse

你的虚拟机包含创建 Bullhorn 所需的所有软件——Oracle Database 12c、SQL Developer 和 Java 1.8——但它不包含 Eclipse IDE。你需要自行下载并安装。幸运的是，安装过程非常简单。

在虚拟机上安装 Eclipse

要在虚拟机上安装 Eclipse，你需要打开 Firefox 并下载 Eclipse 归档文件。然后，你需要从归档文件中提取文件。接下来，你需要运行归档文件中包含的安装程序。最后，打开 Eclipse。

1.  从虚拟机中，打开 Firefox 网页浏览器。你可以通过点击“应用程序”菜单并选择 Firefox 网页浏览器的图标来打开 Firefox。
2.  接下来，访问 [`http://www.eclipse.org/downloads`](http://www.eclipse.org/downloads) 。
3.  点击橙色按钮下载 Eclipse。你将安装最新版本，名为 Oxygen。
4.  你将被重定向到另一个带有橙色“下载”按钮的页面。也点击这个按钮。
5.  当此窗口显示时，选择使用归档管理器打开的选项。归档管理器是用于提取文件的程序。
6.  系统会提示你选择一个文件夹，用于放置提取的文件。浏览到主文件夹并选择“提取”。
7.  勾选打开文件夹以查看文件的选项。看到提取的文件后，查找名为 `eclipse-install` 的文件。
8.  选择安装用于 Java EE 的 Eclipse 的选项。
9.  你的文件夹应为 `/home/oracle/eclipse/jee-oxygen`。
10.  点击“启动”。
11.  接受默认工作空间。你的项目文件将保存在此处。

你已成功安装 Eclipse Oxygen。

如何更改 Tomcat 中的端口

默认情况下，Tomcat 配置为监听以下端口号：8005、8080 和 8009。我们最感兴趣的端口是 8080，因为它用于 HTTP 访问。当你在 Eclipse 中使用 Tomcat 运行 Web 应用程序时，URL 将变为 `http://localhost:8080/ApplicationName`。但是，有时这些端口会被计算机上运行的其他软件占用。幸运的是，Eclipse 可以轻松更改端口号。
