# 2. 什么是数据库？

数据库是一个以安全、高效的方式永久存储数据以供检索的地方。数据库允许我们创建数据并永久保存，允许我们检索之前创建的数据、更新现有数据或删除现有数据。

事务将 SQL 语句分组，使它们作为一个整体应用于数据库。如果某条语句因故失败，则所有语句都会从数据库中撤销。事务确保了数据完整性。事务将 Oracle 这类数据库管理系统与文件系统区分开来。

确保数据有效性的数据库事务属性包括：原子性、一致性、隔离性和持久性。

*   **原子性** 指的是在每个数据库事务中，要么所有信息都被保存，要么所有信息都不被保存。
*   **一致性** 确保事务要么正确执行，要么数据集恢复到事务执行前的状态。
*   **隔离性** 指的是数据库将尚未提交的事务与正在工作的数据库清晰分离的能力。
*   **持久性** 指的是系统保存已提交数据的方式，即使在发生故障和系统重启的情况下，数据也能以其正确的状态可用。

数据库将数据存储在集合中，这些集合通常被视为表。数据库表中的一行代表一条记录，包含相关的属性，这些属性称为字段。字段由表中的列表示。

## 参照完整性

你的数据库管理系统（DBMS）支持参照完整性。参照完整性的目标是避免出现“孤立”数据。当你在表中删除或更新数据时，可能会产生孤立数据；例如，数据库中有一条没有匹配用户的帖子。这种情况不应发生，通常是导入格式错误的数据或无意中删除了用户所致。像 Oracle 这样的数据库管理系统可以通过拒绝会导致孤立数据的更改来强制实施参照完整性，从而防止这种情况。

## 空值

有时，某行中的某一列没有值。在这种情况下，该列存储一个 NULL 值。你可以将其视为一个表示数据缺失的标志。NULL 不同于数值零或长度为零的字符串。它两者都不是。它什么都不是，因为该记录的这一列没有存储任何值。事实证明，NULL 非常有用。你可以在包含 NULL 的记录中搜索字段，并知道这些字段是没有值的。

## 主键、外键和索引

数据库是非常强大的工具，允许我们以惊人的速度搜索和排序数据。数据库机制的一个基本要素是每条记录都应该是唯一的。为了使每条记录唯一，要么使用每个记录都唯一的现有字段，要么为每个记录添加一个包含唯一编号的字段。

你的 DBMS 可以为每条记录生成一个唯一编号。每条记录的唯一编号成为其主键。该字段将该记录与表中的任何其他记录区分开来。

当第二个表包含与第一个表相关的数据时，第二个表可以使用主键字段来引用原始表。

在另一个表中用作引用的主键称为外键。外键定义了一个表（子表）到另一个表（父表）的引用。

为了更高效地访问某一列的数据，数据库会创建一个称为索引的对象。索引为被索引列中的每个值包含一个条目，从而实现对行的快速访问。就像纸质书中的索引一样，数据库可以在索引中查找请求的值，并快速定位其在表中的对应行。

## 连接表

关系是将数据连接到不同表的一种手段。这有助于避免表中的数据冗余。你可以将数据划分到不同的表中——只输入一次——然后通过建立关系从其他表中引用它。

有三种类型的关系，如下所示：

*   **一对一关系**：第一个表中的每条记录在第二个表中只有一条匹配的记录。这通常发生在当一个表中的信息被分割到多个表中时。这种情况并不常见。
*   **一对多关系**：第一个表中的每条记录与第二个表中的多条记录匹配。例如，一个用户可以有多篇帖子。
*   **多对多关系**：第一个表中的每条记录在第二个表中可以有多个对应记录，同时第二个表中的每条记录在第一个表中也可以有多个对应记录。

## 规范化

在规范化的数据结构中，每个表包含关于单个实体的信息，并且每条信息只存储在一个地方。

规范化是在数据库中高效组织数据的过程。这是通过组织关系数据库的列（字段或属性）和表来最小化数据冗余来实现的。

规范化过程的目标是消除冗余数据并确保数据依赖关系合理。



## 结构化查询语言（SQL）

数据库拥有自己的编程语言，即 SQL（结构化查询语言）。SQL 是一种非过程化语言，用于提供数据库访问。所有数据库操作都通过 SQL 执行。Java 可以使用 SQL 与数据库管理系统进行通信。这是通过 JDBC API 实现的，它允许你的应用程序根据各种条件指定要检索哪些记录。例如，你可以选择在特定日期之间创建的记录，或者仅更新满足给定条件的记录。SQL 还可用于创建数据库对象，例如用户表和已保存的查询（即所谓的视图）。视图本身不包含任何数据，它只是存储的查询，使用它可以简化数据访问。

SQL（发音为字母 S-Q-L 或 sequel）是一种专门的数据库语言，由非常接近英语的语句组成。SQL 只有一个目的：与数据库通信。我们与数据库通信以添加、更新或删除数据。我们还与数据库通信以创建和修改表及其他数据库对象。

几乎所有主要的数据库管理系统都支持 SQL。学习 SQL 将使你能够与几乎任何你遇到的数据库进行交互。SQL 易于学习。其语句由描述性的英语单词组成。SQL 功能强大。巧妙地使用语言元素可以让你执行复杂的数据库操作。SQL 是由 ANSI（美国国家标准学会）管理的标准。除了 SQL 的标准实现之外，大多数供应商还实现了自己的专有扩展。你用于 Oracle 的 SQL 版本可能与你用于访问其他数据库的版本不同。
