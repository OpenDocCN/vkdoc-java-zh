# 18. 让页面在所有屏幕尺寸上正常工作

响应式网页设计（RWD）指的是一种开发 Web 应用程序的方法，使其在从台式电脑到手机的任何尺寸屏幕上都能良好显示。电脑、手机或平板电脑的屏幕由像素组成。电脑常见的屏幕分辨率为 1366 × 768。这意味着屏幕宽 1366 像素，高 768 像素。屏幕分辨率决定了文本和图像的显示清晰度。在更高分辨率下，项目显示得更清晰。它们也会显得更小，从而可以在屏幕上容纳更多内容。在平板电脑上查看时，屏幕可能只有 1024 像素宽。手机可能只有 480 像素宽。创建一个能在不同设备上良好显示的网页，被称为使你的网页具有响应性。过去，开发人员实际上会为不同设备创建多个网站。
