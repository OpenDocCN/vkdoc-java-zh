乔纳森·克努森是其他几本 Java 书籍的作者，包括《学习 Java》、《Java 2D 图形》和《Java 加密技术》。他还是《LEGO MINDSTORMS 机器人非官方指南》的作者，但遗憾的是，他未能将这一成功转化为全职事业。乔纳森撰写了大量关于 Java 的文章，以及一些关于 LEGO 机器人的文章。他是四个孩子的父亲，喜欢骑自行车和弹钢琴。更多信息，请访问 [`jonathanknudsen.com/`](http://jonathanknudsen.com/)。
