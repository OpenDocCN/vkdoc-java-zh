# 前言

本书介绍如何使用 Java 技术为移动电话、寻呼机及其他小型设备编程。本书围绕移动信息设备配置文件（MIDP）展开，该配置文件是 Java 2 平台微型版（J2ME）的一部分。本书内容简洁而完整，不仅描述了 MIDP 的全部内容，还深入探讨了 XML 和加密技术等多个高级主题。

第二版涵盖了 MIDP 2.0，其中包含许多令人兴奋的增强功能。新增了三个全新章节，其余部分也经过了细致的更新。

## 读者对象

你阅读本书，很可能是因为你对使用 Java 构建无线应用程序感到兴奋。本书面向已有 Java 编程经验的人士。至少，你应该理解 Java 编程语言和面向对象编程的基础知识。有些章节深入探讨的主题本身就可以写成整本书。如果你想快速了解某个特定主题，这些章节会提供推荐阅读材料。

如果你不熟悉 Java，我建议你阅读一本入门书籍或参加一门课程。《学习 Java（第二版）》（O'Reilly 2002）对于已经熟悉其他语言（如 C 或 C++）的程序员来说，是一本很好的 Java 入门书。



## 本书结构

本书共分为十五章及一个附录，基本包含三个部分。前两章为引言内容。第 3 章至第 12 章全面涵盖了 MIDP API。第 13 章至第 15 章则探讨高级主题。各章节详细划分如下：

*   第 1 章：引言，为后续内容提供背景和动机。本章详细解释了 J2ME，并逐步聚焦到 MIDP。
*   第 2 章：构建 MIDlet，旨在作为引子。本章包含一个示例应用程序，允许你使用任何 MIDP 设备通过互联网查询单词定义。在此过程中，你将学到大量关于为 MIDP 平台开发应用程序的知识。
*   第 3 章：关于 MIDlet 的一切，详细介绍了 MIDP 应用程序的生命周期和打包。本章包含了关于 MIDP 2.0 安全架构的新内容。
*   第 4 章：几乎相同的老内容，描述了 Java 程序员会熟悉的 MIDP API 部分。
*   第 5 章：创建用户界面，是专门介绍 MIDP 用户界面包的几章中的第一章。本章概述了 MIDP 的用户界面包，并详细介绍了简单的可视化组件。
*   第 6 章：列表和表单，接续第 5 章的内容，描述了 MIDP 的高级用户界面组件。
*   第 7 章：自定义项，展示了如何在 MIDP 2.0 中创建你自己的表单项。
*   第 8 章：持久化存储，描述了 MIDP 的数据存储机制。
*   第 9 章：连接世界，包含了关于 MIDP 应用程序如何通过互联网发送和接收数据的所有精彩细节。
*   第 10 章：编程自定义用户界面，描述了可用于专门应用程序用户界面的底层 API。
*   第 11 章：游戏 API，描述了 MIDP 2.0 中用于创建游戏的新特性，包括精灵和瓦片图层。
*   第 12 章：声音与音乐，全面介绍 MIDP 2.0 的新多媒体功能。你将学习如何产生简单音调以及播放采样音频数据。
*   第 13 章：性能调优，描述了应对小型设备上有限资源的技巧。
*   第 14 章：解析 XML，审视了当前可用的一系列小型 XML 解析器。本章描述了如何将解析器移植到 MIDP，并简要讨论了每种解析器的使用方法。
*   第 15 章：保护网络数据，讨论了如何在不安全的互联网上保护有价值的数据。本章包含两个示例应用程序，演示了用于保护数据的加密技术。
*   最后，附录包含了构成 MIDP 的类和接口的完整 API 参考。每个类和接口的公共 API 方法签名均已列出，方便快速查阅。在第二版中，API 参考已做标记，便于识别哪些方法是 MIDP 2.0 中新增的。
