# 后记

虽然我们刚刚开始介绍 Alumni 应用，但目标并非描述整个应用程序，而是描述与 JSF 相关的有趣方面。例如，创建一个最终的类类似于我们为 Books 应用输入数据。原理是相同的，只是内容不同。

当然，可能还有更多 JSF 的内容需要讲解。例如，JSF 2.2 引入了流作用域。¹ 这种作用域有定义的入口和出口点。如果你喜欢定义一系列逻辑耦合的页面（比如向导），它非常有用。我的应用程序都没有使用这个特性。它们的设计方式允许用户随时导航到任何页面，这不适合流作用域。

本书没有关注每一个 JSF 特性，而是专注于通过真实应用程序进行实践开发。你没有经历每一个 JSF 细节，但你学到了（我希望你学到了）很多相关的 Java EE 技术。这些是你创建自己应用程序的基础。我预测，一旦你开始使用 JSF 进行开发，扩展你的知识将会相对容易。如果你需要本书涵盖内容之外的更多信息，请参考 JSF 和 Java EE 参考文档。

我打算在本书变得过于冗长和乏味之前结束它。我希望你喜欢阅读并从中发现价值。
