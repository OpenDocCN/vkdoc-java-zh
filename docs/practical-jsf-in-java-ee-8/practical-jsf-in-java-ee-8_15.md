# 8. TinyCalculator 回顾

Michael Müller^(1 )

(1)德国，北莱茵-威斯特法伦州，布吕尔

到目前为止，我们已经讨论了 TinyCalculator，这是一个为教育目的而开发的小型应用程序。以这个应用程序为例，我们简要介绍了一些技术基础，包括以下内容：

*   HTTP

*   HTML

*   CSS

*   JavaScript

*   Java

*   Maven

我们讨论了一个示例 Servlet，这是实现 JSF 的技术。然后我们继续研究了 JSF 本身，包括以下领域：

*   视图定义语言

*   JSF 生命周期

*   JSF 组件库

*   组件树

*   JSF 标记与 HTML 友好标记

我介绍了一些用于 JSF、Java EE 和 GlassFish 的基本配置文件。最后但同样重要的是，我们介绍了使用 Selenium 进行黑盒或图形用户界面测试。

我们已经从宏观角度审视了 JSF 环境。接下来，我们将转向更大的应用程序，并更深入地探讨使用 Java 和 JSF 进行 Web 开发的各个方面。

TinyCalculator 的源代码可从 [www.apress.com/book/9781484230299](http://www.apress.com/book/9781484230299) 获取。
