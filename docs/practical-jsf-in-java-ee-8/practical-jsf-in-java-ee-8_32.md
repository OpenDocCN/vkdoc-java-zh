# 22. 重温 JSF 生命周期

Michael Müller¹

(1)德国北莱茵-威斯特法伦州布吕尔

让我们回顾一下 JSF 的生命周期。它包含六个阶段，从恢复视图阶段开始。或者，对于新页面，则需要创建视图。图 22-1 应该能唤起你对 JSF 生命周期的记忆。

![A435433_1_En_22_Fig1_HTML.jpg](img/A435433_1_En_22_Fig1_HTML.jpg)

###### 图 22-1 JSF 生命周期

作为视图的主要部分，组件树必须在阶段 1 中（重新）创建。这棵树是 UI 组件的逻辑表示。HTML 组件将在渲染阶段（阶段 6）从组件树中派生出来。但 UI 组件和 HTML 组件之间可能并非一一对应的关系。

除此之外，JSF 标签可以被解释为*标签处理器*或*UI 组件*。标签处理器在创建组件树时被处理，但它本身不会包含在树中。另一方面，UI 组件*会*成为组件树的一部分。并且在渲染阶段，一个组件可能被用来创建多个 HTML 元素。

你可以将标签处理器视为构建树的处理指令，而 UI 组件则是创建输出的处理指令。下一章将使用重复结构的示例来讨论此类问题。
