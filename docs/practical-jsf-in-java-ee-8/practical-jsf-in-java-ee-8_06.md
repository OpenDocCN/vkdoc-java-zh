# 关于技术审校

Manuel Jordan Elera 是一位自学成才的开发者和研究员，他热衷于为自己的实验学习新技术并创建新的集成方案。

Manuel 曾荣获 2010 年 Springy 奖：社区冠军和 2013 年 Spring 冠军。在有限的空闲时间里，他阅读圣经并用吉他作曲。Manuel 以 dr_pompeii 之名广为人知。他为 Apress 审校过多本技术书籍，包括《Pro Spring Messaging》（2017）、《Pro Spring》第四版（2014）、《Practical Spring LDAP》（2013）、《Pro JPA 2》第二版（2013）以及《Pro Spring Security》（2013）。

您可以阅读他撰写的 13 篇关于众多 Spring 技术的详细教程，并通过他的博客 [www.manueljordanelera.blogspot.com](http://www.manueljordanelera.blogspot.com) 联系他。在 Twitter 上关注他：@dr_pompeii。
