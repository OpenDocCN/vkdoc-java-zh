# 4. 编程语言

编程语言最初是一个抽象概念，随后随着时间的推移不断适应和演变。从机器语言的起源、开关输入，延伸到早期像 COBOL 或 BASIC 这样简单的逐行编码，再到 Java 和 C# 等面向对象语言，以及 Q# 等量子计算机语言，编程语言的进化已经历经数十年。

## 编程的起源

19 世纪 40 年代初，查尔斯·巴贝奇提出了一种名为分析机的机器。这仅仅是一个提案——并没有制造出实际的机器，但一位名叫阿达·洛芙莱斯的富有创造力的女性决定撰写一篇文章，详细描述了如何在分析机上表示伯努利数（一种基于数论的递归方程）。这篇文章被认为是世界上第一个计算机程序。

自那时起，可编程设备从理论走向物理，从手动走向自动，从模拟走向数字。随着每一步的进化，我们编程的方式也需要随之演变。

随着大型主机的诞生，数据处理需要向机器发送指令，以处理和解释来自程序员的指令。随后，这些指令被应用于数据，以组织和分析数据。指令通过键盘输入，但由于没有显示器的便利，所有操作都依赖于纸张打印输出，或者以穿孔卡片的形式将数据存储在叠放的卡片组中。如果你仔细观察文本编码和一些编程语言，你会看到诸如“回车”或“打印”之类的术语，这些都是几十年前打印机时代的遗留物。

随着计算机变得更小、更强大，更多的语言被创造出来。语言也被创造出来服务于特定类型的项目和行业，例如数学与科学、数据存储以及图形学。

如今，我们使用的编程语言可以服务于多种不同的目的。事实上，程序员通常需要使用多种编程语言才能完成一个项目。随着语言的演变，它们已经变得专门化，以完成特定的任务。作为一名程序员，你会针对特定任务使用最佳的语言，并将它们组合在一起来创建你的项目。

你今天学习的编程语言在未来将继续演变和变化。随着未来新技术的浪潮，新的语言将被开发出来，让程序员能够推动更多的创新。

## 编程的形式

随着编程在数十年间的演变，你可以进行的编程类型也发生了变化。根据你想要实现的目标，存在不同类型或形式的编程语言，它们以不同的方式工作。作为一名程序员，某些编程形式让你可以直接访问计算机处理器，而另一些则将硬件抽象为更接近人类语言的形式，这种语言需要被翻译或转换成硬件的原生语言。以下是一些你可能遇到的编程形式示例。

### 机器语言

机器语言允许程序员直接向处理器或硬件编写指令。通过向处理器发送由 0 和 1 组成的序列和模式，可以触发操作。因此，程序员输入的代码几乎是原生编写的。汇编语言是机器语言的一种抽象，它使用特殊代码来修改处理器寄存器并执行功能。

### 解释型语言

解释型语言比汇编语言或机器语言更容易被人类阅读。程序员编写代码然后运行它。一个称为解释器的组件会读取每一行代码，然后将其“解释”为计算机的原生指令。这个过程比机器语言慢得多，因为解释器需要转换程序员提供的每一条指令，即使它重复执行同一行代码多次——每次都需要重新解释。JavaScript 就是一种解释型语言的例子。程序员可以停止程序的执行，修改一行代码，然后无需任何其他步骤即可再次运行。

### 编译型语言

编译型语言会获取人类编写的指令，并将该代码发送给一个称为编译器的组件。编译器获取程序指令，将其转换为硬件的二进制字节码或原生代码，并创建一个称为可执行文件的程序。这个程序是硬件和操作系统原生的，并且不容易被转换回原始的编程指令。由于代码现在是计算机的原生格式，它的运行速度比解释型代码快得多，但如果你需要做出更改，就必须调整原始的程序指令并重新编译以创建新的可执行文件。如果你要为多种类型的处理器创建程序，则需要为目标平台编译针对每种原生指令代码的独特版本。C 语言就是一种编译型语言的例子。

### 面向对象语言

面向对象编程（OOP）将一切视为对象。对象可以存储值；执行称为方法的操作；以及接受和返回值。对象使用一个称为类的模板来定义，该类定义了对象可以做什么。然后，程序员可以创建该类的一个实例，该实例拥有该类定义的所有能力。Java 和 C# 就是面向对象语言的例子。

### 数据语言

有些语言是专门为处理数据而设计的。一个例子是 SQL（发音为 see-quel），它是一种为处理数据库而设计的语言。这是一种查询语言，你向数据库提出一个问题，它会返回一组数据作为结果。你可以使用 SQL 将多个数据库组合在一起，创建出可以进行分析的组合。R 是另一种数据语言的例子。R 是为统计计算和图形绘制而设计的。

### 函数式语言

函数式编程以一种非常不同的方式处理编程。可以这样理解：在传统的编程语言（称为命令式语言）中，你是在定义值、对象或组件的状态。你创建并定义要完成的任务（称为算法），这些任务从头到尾执行。函数式编程不受算法的约束。在函数式语言中，你对值执行转换，就像数学中的函数一样。你获取一个值或对象并对其进行修改，并且能够使用函数将多个转换串联起来。函数式语言关注需要做什么，而不太关心如何执行。函数式语言的例子包括 Haskell、Scala 和 F#。

### 脚本语言

操作系统经常需要执行命令来配置服务器、安装软件或执行维护。为了自动化这个过程，存在一些脚本语言，允许像 Windows、Linux 和 macOS 这样的系统将常用命令保存为脚本，该脚本可以多次运行或分发到多台计算机。PowerShell、perl 和 bash 就是脚本语言的例子。

正如你所见，有多种类型的编程语言可以用来执行不同类型的任务。本课程将重点介绍面向对象语言、编译型语言和解释型语言（如 JavaScript、Java、C# 和 Swift）中的概念。
