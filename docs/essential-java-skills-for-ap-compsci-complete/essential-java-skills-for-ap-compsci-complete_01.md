# 1. 引言

计算机科学已成为每个人都需要学习的一项基本生活技能。无论你打算从事商业、技术、创意、建筑还是几乎任何其他领域的职业或副业，你都会发现编程、编码和计算机科学在其中发挥作用。

事实上，如果你查看 LinkedIn 上美国地区的热门技能，在过去一年中，排名前十的技能全部与编程、计算机科学或代码相关。这些技能包括：云计算与分布式计算、统计分析、数据挖掘、移动开发、存储系统管理、用户界面设计、网络与信息安全、中间件与集成软件、Web 架构与开发框架、算法设计以及 Java 开发。^(¹)

计算机科学的独特之处在于，它已不仅仅是一种职业，更成为一项技能。虽然存在“计算机科学家”这样的职位和头衔，但计算机科学的技能，特别是编码和编程，几乎无处不在。

在市场营销中，你需要分析并筛选大量用户数据以及关于人们如何使用你的产品、网站、应用程序或服务的指标。在医学领域，医生和研究人员需要收集并分析来自临床研究的数据，或寻找新的突破。在农业和耕作中，需要部署和管理成千上万的物联网设备，以收集有关土壤条件、湿度和作物健康状况的数据。在建筑和施工中，绘制和分析人们如何使用电梯、建筑基础设施或公共空间，有助于设计出更适合人们工作和生活的建筑。即使在开展副业或“拼搏”时，创业者也可能需要进行研究、搭建网站、编写移动应用或游戏，或执行大量涉及代码、逻辑、自动化和编程的不同活动。

当人们基于代码创建项目时，他们会将工作组织成多个开发块或开发阶段。他们称这些为“冲刺”。因此，我们将以同样的方式学习 Java，使用多个冲刺来教授单个主题，以帮助我们跟上进度。学习编码的难点之一在于，很难找到可以类比的现实世界示例。如果你计划参加计算机科学 AP 考试，许多问题将没有任何现实背景，需要你在没有参考的情况下理解和跟踪代码。

对你来说，这可能会令人困惑，但请记住：编程有两个方面——做了什么以及如何发生。

“做了什么”相当简单，它为代码提供了上下文，例如解析交通数据、掷虚拟骰子或汇总一个季度的销售数据。

然而，“如何发生”则完全是另一回事。这时你需要将事情分解为单个步骤，并开始像计算机一样思考。计算机一次只考虑一个步骤。它不知道下一步是什么，也不知道上一步是什么。它只关心当前。因此，如果它找不到某样东西——你会得到一个错误。如果它试图执行一个不存在的操作——你会得到一个错误。如果你试图按错误的顺序执行操作——你猜对了，你会得到一个错误。

事实上，计算机相当笨。我的意思是，真的非常笨。让它变得聪明的是我们能够将多个操作串联起来，使其发挥作用。随着人工智能的不断进步，计算机本身可以开始进行调整。然后天网被激活，之后会发生什么我们都知道了。

因此，当我们学习编程时，我们将专注于三件事：

*   流程是什么
*   语法是什么
*   流程是什么

流程用流程图表示。我们将学习如何制作这些流程图，以帮助你在编写一行代码之前规划好要做什么。起初，流程图会非常简单，但随后会变得越来越复杂。是的，在一年结束之前，你可能会对流程图感到相当厌烦，但请相信我……它们很有帮助。

语法就是代码；这是你编写的内容，它将你在流程图中创建的流程转换为计算机能够理解的指令。

最后是流程。在这里，你将跟踪代码，观察它沿途存储的数据和信息如何变化，并了解程序的操作如何逐行级联。你将构建能够捕捉编程流程的图表，从而更好地理解计算机如何处理代码，以便更轻松地构思和编写下一个程序。

## 你需要什么

你需要一台 Windows 或 macOS 计算机来完成本书中的项目。你需要安装并配置 Java JDK（Java 开发工具包）以及 JetBrains 出品的 IntelliJ（我们的 IDE 或代码编辑器）。如果你使用的是 macOS，建议卸载内置的 Java JRE（Java 运行时环境），并使用 Oracle Java JDK 中包含的版本。这将避免在构建项目时出现警告。
