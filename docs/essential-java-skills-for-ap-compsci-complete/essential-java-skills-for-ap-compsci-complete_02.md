# 2. 设置 Java JDK 和 IntelliJ

要开始我们的练习，我们需要设置好计算机，使其能够处理 Java 代码并运行它。这被称为创建开发环境。

随着你对编码和编程的深入学习，你会遇到许多不同的工具来帮助你完成工作。事实上，工具如此之多，你可能会发现某个工具比另一个更适合你。虽然它们本质上都做同样的事情，但不同的工具具有不同的功能和选项，能更好地契合不同类型的编码者。



## 编码工具与集成开发环境

编码工具及其更高级的兄弟——集成开发环境（IDE），有时会引发关于哪个才是“最佳”的热烈讨论，但最终，决定权在于你，也就是编码者本人，来选择最适合自己的工具。

为了让本课程以及我们使用的工具易于理解，并在 Windows 和 macOS 系统上保持一致，我们将使用的工具是 **IntelliJ IDEA**。IDEA 是 JetBrains 提供的一款免费编码工具，易于使用，不复杂，并且跨平台体验一致。

本课程中你也可以使用其他工具，例如 Eclipse 或 NetBeans。这些工具提供相同的功能，但为了大家在协作时工具保持一致，我不建议使用它们。

你需要的另一个关键工具是 **Java 开发工具包**（JDK）。为了让 IDE 正常工作，它需要 Java 编译器来生成你的 Java 字节码，然后使用 Java 运行时环境（JRE）在名为 JVM 的虚拟机中运行你的程序。

你将使用的版本是 Java SE（标准版），可以从 Oracle 免费下载。无论你使用 IDEA 还是其他 IDE，都需要安装 JDK 才能编译和构建你的程序。

## 安装与设置

如果你想和我使用相同的工具，你需要安装 JDK 和 IntelliJ IDEA。

要安装我正在使用的工具，你需要一台已安装 Windows 7、8、8.1 或 10 的 PC。

对于 macOS 电脑，你需要 Yosemite、High Sierra 或 Mojave 系统。建议卸载 Apple 默认安装的版本（称为 OpenJDK）。请参考在线说明了解如何操作。

### 安装 JDK

首先，我们需要安装 JDK。

打开浏览器，访问 Oracle 网站：

```
www.oracle.com/technetwork/java/javase/downloads/index.html
```

在此页面，点击 Java JDK 下载按钮。

然后接受许可协议，并选择 Windows（必须安装在 64 位电脑和操作系统上）或 macOS 版本。

浏览器会提示你将安装程序保存到电脑。保存并运行安装程序，使用默认配置。

记住 JDK 的安装位置；稍后你需要用它来将 IntelliJ IDEA 连接到 JDK。

### 安装 IntelliJ

现在我们需要安装 IntelliJ IDEA。

打开浏览器，访问 JetBrains 网站：

```
www.jetbrains.com
```

在“工具”下，找到“IDE”列，选择 IntelliJ IDEA。我们将使用免费的社区版。

浏览器会提示你将安装程序保存到电脑。点击“保存”继续下载。

运行安装程序，接受所有设置的默认值。

大功告成！IntelliJ IDEA 和 Java 开发工具包已成功安装！
