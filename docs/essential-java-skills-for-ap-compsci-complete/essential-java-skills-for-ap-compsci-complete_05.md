# 5. Java 的历史与用途

Java 起步时便怀有远大的目标，而如今，它是全球最流行、使用最广泛的语言之一。



## Java 的起源

Java 始于 Sun Microsystems 公司，并于 1995 年首次向公众发布。该语言的开发始于 1991 年，由 James Gosling、Mike Sheridan 和 Patrick Naughton 主导，其设计目标是与 C 和 C++ 相似，以帮助开发者将他们现有的编程技能应用于这门新语言。Sun Microsystems 于 2009 年被 Oracle 收购。

在开发过程中，Java 最初被称为 Oak，后更名为 Green，最终以 Java 咖啡的名字发布为 Java，这也是为什么 Java 的 logo 是一个冒着热气的咖啡杯。

*一次编写，到处运行*

在发布时，它承诺“一次编写，到处运行”，这意味着你用 Java 创建的程序只需编码一次，就可以在任何支持 Java 的平台上运行。

## Java 的主要目标

从一开始，创建 Java 语言就有五个主要目标：

1.  它必须“简单、面向对象且熟悉”。
2.  它必须“健壮且安全”。
3.  它必须“架构中立且可移植”。
4.  它必须具有“高性能”的执行能力。
5.  它必须是“解释型、多线程且动态的”。

如今，Java 是使用最广泛的语言之一。Java 在 Web 应用程序、使用 JavaFX 等技术的企业桌面应用程序以及 Android 生态系统中的大量应用，使其成为一门值得学习的优秀语言，并且拥有多种灵活的使用方式。

## Java 的用途

自 Java 首次发布以来，它已为电子商务、金融及其他行业的许多大型 Web 应用程序提供了强大支持。Java 应用程序构建在服务器端，并连接到 Web 服务器以处理应用程序的业务逻辑。

Java 也是 Android 的核心语言，使用 Google 的 Android Studio 工具创建的原生应用程序都是用 Java 语言构建的。此外，你还会发现 Java 嵌入在许多设备中，例如电视和其他消费电子产品。

基于 Java 编程语言已经创建了许多框架，以加速跨多个平台和设备的应用程序开发。

对于 Web 应用程序，Struts、JSF、Grails、Spring 等框架可以单独使用或相互组合，以提供可支持数百万用户的可扩展 Web 应用程序。

对于桌面应用程序，JavaFX 等框架允许开发者使用 FXML（一种基于 XML 的标记语言）创建用户界面，并将其与 Java 代码融合。

对于移动端，Google 的 Android SDK 包含一个用于开发 Android 应用程序的工具，以及一个用于在虚拟设备上部署和运行应用程序以进行测试的模拟器。

作为开发者，有许多可用的编码工具供你用于 Java。集成开发环境（IDE）有助于整合你需要执行的许多任务，如编码、代码管理、调试、处理代码仓库以及创建文档。一些比较流行的 IDE 有 IntelliJ IDEA、Eclipse 和 NetBeans。本课程将使用的是 IntelliJ 的 IDE。

Java 为你作为开发者打开了几乎无限的机会，最棒的是，一旦你掌握了基础知识，Java 并不难学或理解。利用你新掌握的 Java 技能，你几乎可以为任何东西构建应用程序。
