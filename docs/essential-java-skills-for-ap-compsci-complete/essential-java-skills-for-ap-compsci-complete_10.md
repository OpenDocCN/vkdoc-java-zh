# 10. 文本字面量与输出

计算机是字面理解事物的。真的，是字面理解。以至于实际上存在一些被称为字面量的东西。

字面意思。

“Hello, World” 就是一个字面量。准确地说，是一个文本字面量。

文本字面量包含字母、符号、标点符号和数字，并由一对引号表示。双引号标记了字面量的开始和结束。

## 文本输出

当你运行像 `println()` 这样的方法时，它会寻找一个叫做字符串的东西。字符串是文本的另一种说法。可以把它想象成串在绳子上的字母珠子序列。

当你向一个使用字符串的方法提供一个字符串时，它会对其进行处理。对于 `println()`，它会将其打印到屏幕上，然后换行。

所以，如果你有以下语句：

```
System.out.println("Hello");
System.out.println("World");
```

这将输出为

```
Hello
World
```

两者显示在两行上。

但还有另一个你可以使用的方法，`print()`。它的工作方式相同，但在文本显示后不会换行。

所以，如果你有以下语句：

```
System.out.print("Hello");
System.out.print("World");
```

这将输出为

```
HelloWorld
```

将两个字符放在同一行。

## 转义序列

文本字面量也可以包含特殊字符。以之前的例子为例：

```
System.out.print("Hello");
System.out.print("World");
```

如果你想在文本字面量本身中包含一个新行，你可以使用两个字符的序列 `\n` 来添加它。

这被称为换行符。它会在你的文本中创建一个换行：

```
System.out.print("Hello\n");
System.out.print("World");
```

这本质上等同于

```
System.out.print("Hello\nWorld");
```

并且在屏幕上显示为：

```
Hello
World
```

那这个呢？

如果你想显示下面这一行，你会怎么做？

```
Mike said, "Hey!"
```

这行文本中包含一个引号，所以如果我们用它创建一个文本字面量，我们会得到

```
"Mike said, "Hey!""
```

但是计算机，再次强调，很笨。所以，它会认为 Hey 中 H 前面的引号是字面量的结束。

这不是我们想要的。

为了解决这个问题，我们可以使用一个转义序列来告诉程序，Hey 周围的引号是字面量的一部分，而不是它的边界：

```
"Mike said, \"Hey\""
```

现在，反斜杠加引号的两个字符序列允许字面量包含引号。

你还可以使用其他转义序列，包括制表符空格序列。

制表符序列 `\t` 会在你的文本中添加一个制表符空格。如果你设置好正确的序列，这有时有助于将内容对齐成列。

就像这样：

```
"First\tMiddle\tLast"
"R.\tDouglas\tWinnie"
```

这将显示为

```
First        Middle            Last
R.           Douglas          Winnie
```

当你将语句上下堆叠时，制表符空格会进行偏移以帮助对齐。
