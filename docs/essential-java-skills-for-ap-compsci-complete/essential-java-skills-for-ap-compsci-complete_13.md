# 13. 注释与空白

嘘。

想对某件事发表评论吗？

这非常简单。

用双斜杠 `//` 开头，之后的所有内容都会被编译器忽略，你就可以随心所欲地写任何内容了。

哦？

你想说更多？

那么，用斜杠星号 `/*` 开始你的长篇大论，然后用星号斜杠 `*/` 结束它。

```
// 单行注释
/* 多行
注释
*/
清单 13-1
单行注释与多行注释
```

哦，想知道一个酷炫的技巧吗？

如果你在一行代码前放一个注释，或者将其包裹成多行注释，就能阻止程序看到这段代码。

这被称为注释掉你的代码。

噢！还有一件事。在 IntelliJ 中有一个快捷键可以做到这一点。只需选中你想要注释掉的代码，然后按 Control-/ 或 Command-/。

很巧妙，对吧？
