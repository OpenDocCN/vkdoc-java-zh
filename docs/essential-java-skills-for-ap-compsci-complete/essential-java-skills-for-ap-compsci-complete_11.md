# 11. 值字面量

计算机存储、处理并访问数据与信息。在计算机程序中，程序员通过变量来完成这些操作。变量具有两面性：对程序员而言，它是你在整个程序中用来引用某个值的名称；而计算机则会在内存中预留一块空间来存储该值，并将其提供给你（作为程序员）使用。

最容易想到的值是数字，比如数字三。三代表事物的数量，例如三个回形针。数字三被称为整数。整数是用于计算对象或事物数量的完整数字。通过整数，我可以对其进行加法或减法运算。

然而，整数分为两种类型：无符号整数和有符号整数。当你想到三个回形针时，你不会考虑正数或负数，你只是简单地拥有三个回形针。但如果我过去有五个回形针，现在只有三个，我该如何表示数值的变化呢？我会使用负数来表示我减少了 –2。

负数可以与其他可正可负的数字一起绘制在数轴上。为了表示这些值，你需要在它们前面加上一个符号来显示正值或负值。尽管大多数情况下正号被省略，但在有符号数中它仍然存在。计算机需要保存这个符号，因此需要为其预留空间。

如果我们正在销售回形针，就需要表示价格。货币是以小数或整数的分数形式来衡量的。因此，回形针的价格可能是 $0.25、€0.25 或 £0.25。小数可能小于一，但小数的细节需要以某种形式在数据中捕获。对于计算机来说，一个数字是一百万还是一百万分之一并不重要，它都需要以相同的方式在内存中存储该值的细节。

## 字面量格式

虽然我们可以查看这些数字并区分它们，但计算机——你猜对了——很笨。除非你明确告诉它数字的格式，否则它无法区分。

字面量是固定值的源代码表示形式。它们直接呈现在代码中，无需任何计算。

我们提供这些值的格式决定了它们的类型。

我们已经遇到过字符串字面量。它由放在一组引号内的字符定义。

还有另一种相关的字母类型，称为字符。它只是用单引号包裹的单个字符：

字符：`'A'`、`'g'`、`'&'`

字符串字面量：`"Hello, world"`、`"Mr. Winnie"`、`"47"`

我们学过的转义序列会被转换为单个字符，这些字符执行独特的输出操作，如制表符空格、换行等。因此我们可以将它们添加到字符串字面量中：

包含引号的字符串：

```
"Janet said, \"Hello!\""
```

包含换行的字符串：

```
"Line one\nLine two"
```

对于数字，情况则有所不同。数字有两种基本类型：整数和小数。

整数是完整的数字，永远不能包含任何小于一的小数部分。小数则可以包含整数的分数部分。通常，我们将它们表示为不带小数点的数字或带小数点的数字：

整数：`23`、`55`、`1701`

小数：`23.0`、`3.14`、`47.001`

不过，小数有两种类型。（计算机相关的事情没有简单的，对吧？）而这归根结底都与内存有关。

想象一下，你有一个用于测量两把椅子之间距离的小数。你可能可以用整数来测量，比如 15 英寸。

想象一下，如果你需要测量电路板上两根导线之间的距离。你需要更高的精度，可能是 0.0005 英寸。

那么，如果你需要测量分子中两个原子之间的距离呢？以一氧化碳为例。碳原子和氧原子之间的距离是 1.13×10¹⁰ 米，即 0.000000000113 米。这需要更高的精度才能准确。

精度是数量的镜像。你拥有的东西越多，所需的内存就越多。同样，你需要越精确，所需的内存就越多。

但有时，你并不立即需要那么高的精度。那么我们如何将其表示为值字面量呢？

我们给数字添加后缀。

小数有两种基本的精度：单精度浮点数和双精度浮点数。为了区分它们，你在末尾添加一个后缀。对于单精度浮点数，你添加 `f`。对于双精度浮点数，你添加 `d`：

单精度浮点数小数数字字面量：`65.4f`

双精度浮点数数字字面量：`65.4d`

这将允许这些字面量以不同的精度级别进行管理。通常，除非你在医学或科学领域工作，否则对于大多数应用程序，你无需担心这些。此外，缺少后缀的数字被视为双精度浮点数。
