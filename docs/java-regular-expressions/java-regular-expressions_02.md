# 引言

任何计算机语言的基本目标都是操作数据。传统上，只要数据以对象形式表示，Java 就是实现这一目标的优秀语言。然而，Java 的原始数据操作机制一直有所欠缺，尤其是与 Perl 和 awk 等语言提供的强大处理能力相比。

在 Java 2 标准版（J2SE）中引入标准的正则表达式包，是纠正这一疏忽的绝佳一步。`java.util.regex` 包为开发者提供了在 Java 中使用正则表达式所需的一切，并且全部封装在易于使用的面向对象结构中。我相信你会发现，`java.util.regex` 包不仅能成为你编程武器库中极其强大的工具，也是一件你会乐于使用的优雅工具。掌握它之后，你会像我一样好奇，以前没有它时是如何应对的。

## 本书内容

本书是对 J2SE 内置正则表达式支持的全面介绍，旨在帮助那些几乎没有正则表达式经验的 Java 程序员。它既是一本参考书，也是一本解释性文本。虽然具备正则表达式背景会有所帮助，但在呈现材料时，我并未做此类假设。核心目标是帮助日常程序员解决日常问题。

阅读本书后，你应该能够通过使用 Java 内置的正则表达式支持，快速高效地解决大量常规的文本验证、搜索、修改和替换问题。当然，本书也涵盖了一些正则表达式的高级特性。读完第 1 章后，你应该就能有效地运用你对正则表达式的新理解了。

## 读者对象

如果你刚接触正则表达式，但对 Java 语言很熟悉，那么本书就是为你准备的。如果你有正则表达式背景，但需要一本 Java 正则表达式包的参考书，你也会发现本书很有用。然而，如果你是 Java 新手，你可能会发现先阅读一些入门书籍会更好。市面上有很多优秀的入门书籍，不过我的推荐是 Kathy Sierra 和 Bert Bates 合著的 *Head First Java*（O'Reilly & Associates, 2003）以及 Bruce Eckel 的 *Thinking in Java, Third Edition*（Prentice Hall, 2002）。这两本书都不会让你失望。

## 本书结构

本书包含五章和三个附录。它旨在提供循序渐进的学习体验，因此各章节内容层层递进。我在以下各节中描述了各章和附录的内容。

### 第 1 章

本章介绍正则表达式，并提供一些简单的示例和解释来帮助你入门。它探讨了 J2SE 正则表达式的语法、操作，以及与你可能已经熟悉的其他语言正则表达式的差异。本章还提供了一个正则表达式教程。

即使你有正则表达式背景，我也建议你浏览本章中的示例——正则表达式有很多不同的“风味”，它们之间很少有同构的映射关系。如果你是 J2SE 正则表达式的新手，或者需要全面复习正则表达式，第 1 章 是一个自然的起点。

### 第 2 章

第 2 章 通过 `Pattern` 和 `Matcher` 类介绍了 Java 对正则表达式的内置支持。每个方法和属性都得到了详细剖析，除了最琐碎的方法外，都附有配套的代码示例来突出其正确用法。第 2 章 涵盖了哪些设置和标志可能会影响代码效率。此外，第 2 章 详细介绍了 `String` 类上支持正则表达式的五个新方法，并就其正确用法提供了建议和示例。

### 第 3 章

第 3 章 阐述了一些高级正则表达式概念，包括组、非捕获组、贪婪限定符、积极限定符、懒惰限定符、占有限定符、正向预查、负向预查、正向回顾和负向回顾。它提供了大量示例和详细解释，说明这些概念如何工作，以及它们如何在日常任务中为你带来好处。

### 第 4 章

第 4 章 提供了在 Java 面向对象环境中使用正则表达式的建议和提示。本章涵盖了最佳实践、示例以及从类似包中汲取的经验教训。

### 第 5 章

第 5 章 提供了大量功能完备的示例，并附有解释和代码，这些示例建立在前面章节所介绍的材料之上。第 5 章 旨在说明我在尝试用 Java 解决正则表达式问题时使用的开发过程。

### 附录 A

附录 A 提供了正则表达式语法和各种正则表达式元字符的完整列表及解释。

### 附录 B

附录 B 提供了 Java 中 `Pattern` 和 `Matcher` 类的方法摘要。



### 附录 C

附录 C 提供了简单的正则表达式，不附带代码，用于帮助处理日常常见任务，例如验证电子邮件、检查文本格式、提取值等。
