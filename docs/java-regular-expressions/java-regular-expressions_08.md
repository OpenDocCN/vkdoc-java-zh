# 附录 C：常用正则表达式模式

本附录提供了一些实用的正则表达式模式，可用于常见的匹配和验证任务。

表 C-1：IP 地址 *^(([0-1]?\d{1,2}\.)|(2[0-4]\d\.)|(25[0-5]\.)){3} (([0-1]?\d{1,2})|(2[0-4]\d)|(25[0-5]))*$

| **正则表达式** | **描述** |
| --- | --- |
| **^** | 行首 |
| **(** | 一个组，包含 |
| **(** | 一个子组，包含 |
| **[0-1]?** | 零或一个（均可选），后跟 |
| **\d** | 任意数字 |
| **{1,2}** | 重复一次或两次，后跟 |
| **\.** | 一个句点 |
| **)** | 关闭子组 |
| **&#124;** | 或 |
| **(** | 一个子组，包含 |
| **2** | 数字 *2*，后跟 |
| **[0-4]** | 从 *0* 到 *4* 的任意数字，后跟 |
| **\d** | 任意数字，后跟 |
| **\.** | 一个句点 |
| **)** | 关闭子组 |
| **&#124;** | 或 |
| **(** | 一个子组，包含 |
| **2** | 数字 *2*，后跟 |
| **5** | 数字 *5*，后跟 |
| **[0-5]** | 从 *0* 到 *5* 的任意数字，后跟 |
| **\.** | 一个句点 |
| **)** | 关闭子组 |
| **)** | 关闭组 |
| **{3}** | 重复三次，后跟 |
| **(** | 一个组，包含 |
| **(** | 一个子组，包含 |
| **[0-1]?** | 零或一个（均可选），后跟 |
| **\d{1,2}** | 任意两位数字 |
| **)** | 关闭子组 |
| **&#124;** | 或 |
| **(** | 一个子组，包含 |
| **2** | 数字 *2*，后跟 |
| **[0-4]** | 从 *0* 到 *4* 的任意数字，后跟 |
| **\d** | 任意数字 |
| **)** | 关闭子组 |
| **&#124;** | 或 |
| **(** | 一个子组，包含 |
| **2** | 数字 *2*，后跟 |
| **5** | 数字 *5*，后跟 |
| **[0-5]** | 从 *0* 到 *5* 的任意数字，后跟 |
| **)** | 关闭子组 |
| **)** | 子组 |
| **$** | 行尾 |
| * **中文解释：** 由句点分隔的三组三位数字，每组范围从 0 到 255，后跟三位数字，每位范围从 0 到 255。 |

|  | 注意  | 以下模式也能匹配 IP 地址，但所有组都被标记为非捕获组： 
```
(?:(?:[0-1]?\d{1,2}\.)&#124;(?:2[0-4]\d\.)&#124;(?:25[0-5]\.)){3}(?:(?:[0-1]?\
d{1,2})&#124;(?:2[0-4]\d)&#124;(?:25[0-5]))
```

这比之前的模式效率稍高，但可读性较差。 |

表 C-2：简单电子邮件 *^(\p{Alnum}+(\.|\_|\-)?)*\p{Alnum}@(\p{Alnum}+ (\.|\_|\-)?)*\p{Alpha}*$

| **正则表达式** | **描述** |
| --- | --- |
| **^** | 行首 |
| **(** | 一个组，包含 |
| **\p{Alnum}** | 一个字母或数字 |
| **+** | 重复一次或多次，后跟 |
| **(** | 一个子组，包含 |
| **\.** | 一个句点 |
| **&#124;** | 或 |
| **\_** | 一个下划线 |
| **&#124;** | 或 |
| **\-** | 一个连字符 |
| **)** | 关闭子组 |
| **?** | 前面的标点符号是可选的 |
| **)** | 关闭组 |
| ***** | 前面的组可以重复零次或多次， |
|   | 后跟 |
| **\p{Alnum}** | 一个字母或数字，后跟 |
| **@** | 一个 *@* 符号，后跟 |
| **\p{Alnum}** | 一个字母或数字 |
| **+** | 重复一次或多次，后跟 |
| **(** | 一个组，包含 |
| **\.** | 一个句点 |
| **&#124;** | 或 |
| **\_** | 一个下划线 |
| **&#124;** | 或 |
| **\-** | 一个连字符 |
| **)** | 关闭组 |
| **?** | 前面的标点符号是可选的 |
| **\p{Alpha}** | 一个大写或小写字母 |
| **$** | 行尾 |
| * **中文解释：** 任意数量的字母数字字符，后跟单个连字符、句点或下划线，但以字母数字字符结尾；后跟一个 *@* 符号；后跟任意数量的字母数字字符；后跟单个连字符、句点或下划线，但以大写或小写字符结尾。 |

|  | 注意  | 以下模式与上一个模式完全匹配，但额外允许使用 IP 地址： 
```
^(\p{Alnum}+(\.&#124;\_&#124;\-)?)*\p{Alnum}@(((\p{Alnum}+(\.&#124;\_&#124;\-)?)*
\p{Alpha})&#124;((([0-1]?\d{1,2}\.)&#124;(2[0-4]\d\.)&#124;(25[0-5]\.)){3}(([0-1]?\d{1,2})&#124;
(2[0-4]\d)&#124;(25[0-5]))))$ 
```

有关 IP 地址模式的详细分解，请参见表 C-1。 |

表 C-3：恰好重复 n 次的数字 *\d{n}*，其中 n 是所需数字的个数

| **正则表达式** | **描述** |
| --- | --- |
| **\d** | 任意数字 |
| **{n}** | 重复 *n* 次 |
| * **中文解释：** *n* 个数字。因此，如果 *n* 等于 4，则匹配任意四位数字。 |

表 C-4：恰好重复 n 次的字符 *\w{n}*，其中 n 是所需字符的个数

| **正则表达式** | **描述** |
| --- | --- |
| **\w** | 任意数字、任意数字或下划线符号 |
| **{n}** | 重复 *n* 次 |
| * **中文解释：** *n* 个字符。因此，如果 *n* 等于 4，则匹配任意四个字符。 |

表 C-5：重复 n 到 m 次的字符 *\w{n.m}*，其中 n 是所需字符的个数

| **正则表达式** | **描述** |
| --- | --- |
| **\w** | 任意数字、任意数字或下划线符号 |
| **{n** | 重复 *n* 次 |
| **m}** | 但不超过 *m* 次 |
| * **中文解释：** *n* 个字符。因此，如果 *n* 等于 4 且 *m* 等于 9，则匹配任意四、五、六、七、八或九个字符。 |

表 C-6：信用卡：Visa、MasterCard、American Express 和 Discover *^((4\d{3})|(5[1-5]\d{2})|(6011))-?\d{4}-?\d{4}-?\d{4}|3[4,7]\d{13}*$

| **正则表达式** | **描述** |
| --- | --- |
| **^** | 行首 |
| **(** | 一个组，包含 |
| **(** | 一个子组，包含 |
| **4** | 数字四，后跟 |
| **\d[3]** | 任意三位数字 |
| **)** | 关闭子组 |
| **&#124;** | 或 |
| **(** | 一个子组，包含 |
| **5** | 数字 *5*，后跟 |
| **[1-5]** | 从 *1* 到 *5* 的任意数字，后跟 |
| **\d[2]** | 任意两位数字 |
| **)** | 关闭子组 |
| **&#124;** | 或 |
| **(** | 一个子组，包含 |
| **6001** | 数字 *6*、*0*、*0*、*1* |
| **)** | 关闭子组，后跟 |
| **-?** | 一个可选的连字符，后跟 |
| **\d{4]** | 任意四位数字，后跟 |
| **-?** | 一个可选的连字符，后跟 |
| **\d{4]** | 任意四位数字，后跟 |
| **-?** | 一个可选的连字符，后跟 |
| **\d{4]** | 任意四位数字，后跟 |
| **-?** | 一个可选的连字符 |
| **&#124;** | 或 |
| **3** | 数字 *3*，后跟 |
| **[4,7]** | 一个 *4* 或 *7*，后跟 |
| **\d{13]** | 任意十三位数字，后跟 |
| **$** | 行尾 |
| * **中文解释：** 以 *4* 开头后跟三位数字，或以 *5* 开头后跟三位数字，或以 *6011* 开头，后跟一个连字符，再后跟三组四位数字；或者以 34 开头后跟十三位数字，或以 37 开头后跟十三位数字。 |

|  | 注意  | 此正则表达式无法且不能符合模 10 验证。要查找执行此验证的 Java 程序，请访问 [`www.influxs.com`](http://www.influxs.com)。 |

表 C-7：实数 *^[+-]?\d+(\.\d+)?*$

| **正则表达式** | **描述** |
| --- | --- |
| **^** | 行首，后跟 |
| **[+-]?** | 一个可选的加号或减号 |
| **\d+** | 后跟一个或多个数字，后跟 |
| **(** | 一个组，包含 |
| **\.** | 一个句点，后跟 |
| **\d+** | 一个或多个数字 |
| **)?** | 关闭组，并将其设为可选 |
| **$** | 行尾 |
| * **中文解释：** 任意数量的数字，后跟一个可选的小数部分。 |
