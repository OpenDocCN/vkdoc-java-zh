# 前言

本书探讨为基于 Java 的应用程序构建灵活且健壮的应用程序日志框架的概念。在介绍应用程序日志之后，本书涵盖了两种最流行的基于 Java 的日志 API：JDK 1.4 日志 API 和 Apache log4j 1.2.6 版本。本书对这两种 API 进行了全面的研究，并在最后一章中，您将找到这两种日志 API 的比较研究以及一些使用它们的最佳实践。

本书旨在提供一份关于使用 JDK 1.4 日志 API 和 Apache log4j 的深入指南，重点在于：

*   理解这些 API 的内部机制及其工作原理
*   使用这些 API 中极其有用且健壮的日志功能
*   针对某些特定的、应用程序特有的需求，扩展现有的日志框架

本书提供了大量用 Java 语言编写的示例，从简单程序到复杂程序。简单的示例解释了日志 API 的工作原理以及它们在日志输出方面能够产生什么。复杂的示例类似于现实生活中的应用场景，并演示了日志对于维护大规模分布式应用程序组件是多么有用。最后，JDK 1.4 日志 API 和 Apache log4j 之间的比较将作为指导，帮助您决定在项目中使用所讨论的两种 API 中的哪一种。

## 目标读者

本书讨论基于 Java 语言的日志 API，面向从初级到高级的 Java 语言应用程序开发者。它提供了许多简单直接的示例，只需要对 Java 语言有最基本的了解。因此，本书对非 Java 语言应用程序开发者也有帮助，他们仍然可以从本书提出的理念中受益。

## 章节概要

本书主要分为两部分：一部分介绍 JDK 1.4 日志 API，另一部分介绍 Apache log4j 1.2.6 版本。只对某一特定 API 感兴趣的读者可以跳过讨论另一 API 的章节。总的来说，本书包含以下章节：

### 第 1 章：应用程序日志简介

本章讨论应用程序日志。它描述了与应用程序日志各种概念相关的优缺点。本章还讨论了一个好的日志框架应具备的几个标准。

### 第 2 章：JDK 1.4 日志 API

本章介绍 JDK 1.4 日志 API，并深入讨论该 API 的内部机制和使用方法。

### 第 3 章：格式化 JDK 1.4 日志信息

本章探讨与日志输出格式化相关的问题。它讨论了 JDK 1.4 日志 API 中可用的格式化选项，并展示了如何编写自定义格式化程序以进行定制化格式化。

### 第 4 章：扩展日志框架

本章专门解释和演示如何扩展现有的 JDK 1.4 日志框架，以编写自定义的、特定于应用程序的日志组件。它提供了一些现实生活中的示例，说明如何开发和自定义日志组件。

### 第 5 章：理解 Apache log4j

本章基于 1.2.6 版本深入讨论 Apache log4j。它解释了 log4j 框架中涉及的核心对象，以及它们如何工作和相互交互，最终产生日志输出。

### 第 6 章：在 log4j 中格式化日志信息

本章讨论 log4j 中可用的不同格式化选项，以构建最终的日志输出。它还展示了如何使用可配置的模式来格式化日志信息。

### 第 7 章：使用 log4j 进行高级日志记录

本章专门介绍 log4j 的高级日志功能。它详细介绍了通过数据库、Java 消息服务（JMS）、简单邮件传输协议（SMTP）以及 log4j 中可用的其他各种选项来分发日志信息的过程。

### 第 8 章：扩展 log4j 以创建自定义日志组件

本章重点介绍扩展现有 log4j 框架以编写自定义的、特定于应用程序的日志组件的技术。它提供了一些现实生活中的示例，演示了自定义日志组件在某些应用场景中是多么有效。



### 第 9 章：使用 Apache 日志标签库

本章讨论 Apache 日志标签库，该库可与 Java 服务器页面（JSP）配合使用，实现基于 log4j 的日志记录活动。本章介绍日志标签库的安装与使用，并展示如何在该标签库中整合自定义标签。

### 第 10 章：最佳实践

本章提供使用 JDK 1.4 日志 API 和 Apache log4j 的最佳实践指南，并对这两种日志 API 进行了比较研究。
