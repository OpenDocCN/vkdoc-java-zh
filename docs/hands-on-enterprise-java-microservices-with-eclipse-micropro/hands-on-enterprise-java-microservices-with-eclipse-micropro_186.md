# GraphQL

GraphQL 是一种用于 API 的开源数据查询与操作语言，也是一个利用现有数据完成查询的运行时。它能够解析来自客户端的字符串，并以可理解、可预测且预定义的方式返回数据。GraphQL 是 REST 的一种替代方案，但并非必然取代后者。MicroProfile GraphQL 规范的目标是提供一组 API，使开发者能够用 Java 快速开发可移植的基于 GraphQL 的应用程序。作为 MicroProfile 项目的一部分，在撰写本文时，MicroProfile GraphQL 规范仍处于提议或草案阶段。

GraphQL 基于 Facebook 的一项规范。有关更多信息，请参阅 `https://graphql.github.io/graphql-spec`。关于 GraphQL 的更全面概述，可访问 [`graphql.org/`](https://graphql.org/)。

GraphQL 与 REST 有许多相似之处，两者都广泛用于现代基于微服务的应用程序中。
