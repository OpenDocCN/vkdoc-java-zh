# Apache TomEE

Tomitribe 是开源项目 TomEE 的赞助商，该项目实现了 Eclipse MicroProfile 规范。Apache TomEE 是在 Apache Tomcat 的基础上添加了 Java EE 功能而构建的。TomEE 通过了 Java EE 6 Web Profile 认证。正如其 GitHub 描述的那样，*Apache TomEE 是一个轻量级但功能强大的 JavaEE 应用服务器，拥有丰富的工具集*。你可以下载几个不同版本的 TomEE；例如，TomEE、TomEE+ 和 TomEE WebApp，但我们感兴趣的是 TomEE MicroProfile。对于 MicroProfile，TomEE 会为你的微服务生成一个 uberjar，你可以按如下方式运行它：

```
$ java -jar <可执行 JAR 文件>
```
