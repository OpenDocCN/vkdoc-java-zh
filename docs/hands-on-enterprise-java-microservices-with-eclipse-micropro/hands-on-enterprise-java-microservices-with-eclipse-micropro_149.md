# MicroProfile 互操作性 – 会议应用程序

**会议应用程序** 首次于 2016 年 11 月在比利时 Devoxx 大会上展示（[`www.youtube.com/watch?v=iG-XvoIfKtg`](https://www.youtube.com/watch?v=iG-XvoIfKtg)），是一个 MicroProfile 演示，展示了不同 MicroProfile 供应商实现的集成和互操作性。这很重要，因为它展示了规范中实现与接口的分离，提供了一个平台，允许供应商开发并提供他们自己的实现，这些实现可以与其他竞争性实现共存。所有实现共有的公共接口也为最终用户提供了灵活性和选择权，可以使用任何 MicroProfile 实现……
