# MicroProfile OpenAPI 与类型安全 REST 客户端

Eclipse MicroProfile 为 Java 微服务提供了一套丰富的规范。其中，Eclipse MicroProfile OpenAPI 和 Eclipse MicroProfile REST Client 分别用于帮助编写微服务的 API 文档，并提供在 REST 端点上进行类型安全调用的 API。OpenAPI 简化了微服务端点的文档编写，并使这些元数据可供第三方开发者查阅。类型安全的 REST 客户端则简化了对象与 HTTP 或 JSON 之间的序列化与反序列化。

本章将涵盖以下主题：

*   每个规范所提供的功能
*   部分功能的简单代码示例
*   如何获取关于每个规范的更多信息
