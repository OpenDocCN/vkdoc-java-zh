# 第 1 章

1.  一个企业级 Java 微服务具有以下特征：

    *   使用 Java 语言编写。
    *   可以使用任何 Java 框架。
    *   可以使用任何 Java API。
    *   必须达到企业级标准：可靠、可用、可扩展、安全、健壮且高性能。
    *   必须满足 [`martinfowler.com/microservices/`](https://martinfowler.com/microservices/) 中列出的微服务架构特征，具体如下：
        *   通过服务进行组件化
        *   围绕业务能力组织
        *   产品而非项目
        *   智能端点和哑管道
        *   去中心化治理
        *   去中心化数据管理
        *   基础设施自动化
        *   为故障而设计
        *   演进式设计
