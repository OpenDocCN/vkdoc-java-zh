# 第 4 章

1.  在仅通过 HTTP 状态码来确定 HC 状态的环境中，这种线路格式是不可用的。
2.  MP-HC 响应可以使用 `HealthCheckResponse#withData()` 方法包含任意属性。
3.  您可以为每个服务创建一个 `HealthCheck` 实现，MP-HC 功能将逻辑上组合它们以生成整体状态响应。
4.  JWT 是 JSON Web Token，一个具有头部、载荷和签名组件的 JSON 格式对象。
5.  声明是来自 JWT 载荷的单个命名值。
6.  任何可以用 JSON 表示的内容都可以在 JWT 中使用。
7.  验证 JWT 的主要步骤是确认它是基于配置的公钥通过 RS256 算法签名的。
8.  除了 groups 声明之外，还可以查看其他声明以进行检查，从而添加特定于应用程序的授权决策。
