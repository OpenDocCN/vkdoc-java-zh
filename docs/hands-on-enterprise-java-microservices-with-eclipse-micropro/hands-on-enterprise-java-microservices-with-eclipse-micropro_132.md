# 总结

在本章中，我们学习了两个 Eclipse MicroProfile 规范，即 Eclipse MicroProfile OpenAPI 和 Eclipse MicroProfile REST Client。前者提供了为微服务生成符合 OpenAPI 规范的文档的规范，后者则提供了以类型安全的方式调用 REST 端点的规范。在本章中，我们介绍了这些规范的具体功能，提供了一些示例代码，并提供了如何获取这些规范更多信息的指引。您已经了解了 Eclipse MicroProfile OpenAPI 和 Eclipse MicroProfile REST Client 规范的特性和功能，如何使用它们的注解和编程接口，以及如何将它们集成到您的应用程序中。
