# MicroProfile 实现、Quarkus 以及通过会议应用实现互操作性

Eclipse MicroProfile 的优势之一在于它提供了一个规范，允许许多实现彼此互操作。这一优势鼓励了许多供应商和社区团体将 Eclipse MicroProfile 规范实现为开源实现。目前市场上有八个 Eclipse MicroProfile 实现，第九个实现 Quarkus 也已加入。

本章将涵盖以下主题：

*   描述八个 Eclipse MicroProfile 实现以及如何获取每个实现的更多信息
*   如何为这些实现中的每一个生成 Eclipse MicroProfile 示例代码……
