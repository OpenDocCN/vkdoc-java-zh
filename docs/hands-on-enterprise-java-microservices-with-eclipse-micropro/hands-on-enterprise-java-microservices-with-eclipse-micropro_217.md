# 第 5 章

1.  分布式追踪提供了请求从端到端所发生情况的微观视图，而指标则暴露单个组件的标量数值。
2.  分布式追踪系统通常提供诸如根本原因和关键路径分析、上下文日志记录、分布式上下文传播以及服务依赖关系图等功能。
3.  自动被追踪的是 JAX-RS 服务端点和 MicroProfile Rest 客户端。一些供应商也可以自动追踪 JAX-RS 客户端。
4.  对于每个 REST 请求，都会添加这些标签：`http.method`、`http.status_code`、`http.url`、`component`、`span.kind`，如果抛出异常，还会添加 `error`。
5.  可以通过使用 `@Traced` 注解或注入追踪器并创建...来添加显式仪表化。
