# Eclipse MicroProfile 简介

Eclipse MicroProfile 是一套用 Java 语言编写的微服务规范。它是一个由社区驱动、市场上有多种实现的项目。该项目于 2016 年 6 月首次宣布，持续开发一套通用的**应用程序编程接口**（**API**），用于实现适用于现代应用开发技术、架构和环境的 Java 微服务。在本章中，你将了解 Eclipse MicroProfile 的起源和重要性。
