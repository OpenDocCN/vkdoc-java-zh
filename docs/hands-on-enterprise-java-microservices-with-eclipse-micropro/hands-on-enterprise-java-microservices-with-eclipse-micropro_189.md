# 沙箱中的项目

MicroProfile 项目沙箱是社区成员通过尝试实现某些特性和功能来提出想法的地方，旨在从社区成员那里获取反馈、讨论和评估，以决定该想法是否应成为 MicroProfile 项目的新 API/规范。

MicroProfile 沙箱位于 [`github.com/eclipse/microprofile-sandbox`](https://github.com/eclipse/microprofile-sandbox)。

过去从沙箱毕业并成为官方 MicroProfile 项目的提案包括 GraphQL 和 Reactive Streams Operators。在撰写本文时，沙箱中只有一个提案项目：Boost。
