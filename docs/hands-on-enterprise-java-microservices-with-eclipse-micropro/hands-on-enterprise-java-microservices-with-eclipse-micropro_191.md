# Eclipse MicroProfile 与 Jakarta EE

Eclipse Jakarta Enterprise Java 项目与 Eclipse MicroProfile 项目之间是什么关系？简短的回答是：有待观察。详细的回答是：让我们从头说起。

Eclipse MicroProfile 项目的启动是为了解决在 **Java 社区流程**（**JCP**）下 Java EE 规范进展缓慢的问题。

有关 Java 社区流程的更多信息，请访问 `https://jcp.org/en/home/index`。

自 MicroProfile 项目迁移到 Eclipse 基金会以来，已经过去了两年多。大约一年后，Oracle 宣布打算将 Java EE 迁移到 Eclipse 基金会，并将其重命名为 Jakarta EE。向 Jakarta 的迁移是一个旷日持久的过程，目前仍在进行中...
