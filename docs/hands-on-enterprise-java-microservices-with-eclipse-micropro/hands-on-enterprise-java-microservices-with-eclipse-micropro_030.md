# MicroProfile Starter

MicroProfile Starter 是一个示例源代码生成器，其目标是通过在 Maven 项目中生成可工作的示例代码，帮助开发者快速上手使用和利用社区驱动的企业级 Java 微服务开源规范 Eclipse MicroProfile 的功能。

自 2016 年中该项目创建以来，MicroProfile Starter 的想法就已存在，并在 2016 年 Devoxx BE（2016 年 11 月 7 日那周）上进行了公开讨论。在可用的前两周内，世界各地的开发者通过 MicroProfile Starter 项目创建了超过 1200 个项目，这很好地积极表明了其在全球范围内的采用情况。
