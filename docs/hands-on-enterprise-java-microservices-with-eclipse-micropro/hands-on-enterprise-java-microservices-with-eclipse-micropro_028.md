# 开放贡献的沙盒方法

为潜在的 MicroProfile 子项目创建工作组时，也可能分配一个沙盒，这是 MicroProfile 社区提供的另一个用于尝试新想法的资源。沙盒仓库是一个位于 [`github.com/eclipse/microprofile-sandbox`](https://github.com/eclipse/microprofile-sandbox) 的 GitHub 仓库，用于孵化最终将转变为新规范独立仓库的想法和代码示例。任何人都可以发起拉取请求，并使用沙盒来试验新想法、共享代码和文档，这些内容可作为社区 Google Group、MicroProfile Hangout 会议或工作组会议讨论的一部分。请保持您的拉取请求开放...
