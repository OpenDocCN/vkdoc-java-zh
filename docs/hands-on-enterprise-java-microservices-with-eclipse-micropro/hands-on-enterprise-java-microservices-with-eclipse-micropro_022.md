# Eclipse MicroProfile 介绍

Java EE 一直是一个极其成功的平台。**Java 社区流程**（**JCP**）在其近 20 年的历史中管理了超过 20 个兼容实现，造就了一个价值 40 亿美元的产业。然而，Oracle 对 Java EE 的管理（无论有意与否）阻碍了创新，尽管其他标准已经发展起来，但全球 Java 社区和所有主要企业的 CIO 都渴望在企业内部拥有一个开放的 Java 标准。

在其早期阶段，J2EE 从 J2EE 1.2 到 J2EE 1.4 的发展速度相对较快，因为该平台需要满足企业的即时需求。从 2006 年 5 月的 Java EE 5 开始，发展速度开始...
