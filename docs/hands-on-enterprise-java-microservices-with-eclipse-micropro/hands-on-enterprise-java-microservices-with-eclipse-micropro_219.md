# 第 7 章

1.  在撰写本文时，共有八个 Eclipse MicroProfile 的实现，它们都是开源的。分别是 Thorntail、Open Liberty、Apache TomEE、Payara Micro、Hammock、KumuluzEE、Launcher 和 Helidon。此外，还有最新加入的 Quarkus。
2.  应用服务器是 Java EE 应用程序的容器。应用组装器仅包含应用程序所需的功能，而不是要求应用服务器启动并运行，并且通常会生成一个可执行的 JAR 文件。应用组装器可以生成一个 uberjar（一个自包含的可运行 JAR 文件），或者一个应用程序 JAR，其运行时依赖项位于子目录中，例如，一个附带的 `lib` 或 `libs` 子目录。……
