# 总结

在本章中，我们了解了市场上目前存在的开源 MicroProfile 实现，它们属于哪种类型，如何获取关于每种实现的更多信息，以及如何使用 MicroProfile Starter 为这些实现生成示例代码。我们还介绍了最新的 MicroProfile 实现新成员 Quarkus，它在解释模式和编译模式下显著改善了 Java 的启动时间和内存消耗，进一步改进了用于云原生微服务和无服务器环境的 MicroProfile。你还了解了会议应用程序，它展示了 MicroProfile 在不同实现之间的互操作性。

作为 Eclipse MicroProfile 的使用者，并且由于其跨实现互操作性的特性，你可以自由选择最适合你组织或最符合你环境的实现，最终让你能够为正确的任务选择正确的工具。此外，你无需为了获得商业支持的 Eclipse MicroProfile 版本而局限于单一供应商，因此，你有优势根据自己的条件进行协商，并从不同供应商提供的丰富 MicroProfile 功能集中进行选择。
