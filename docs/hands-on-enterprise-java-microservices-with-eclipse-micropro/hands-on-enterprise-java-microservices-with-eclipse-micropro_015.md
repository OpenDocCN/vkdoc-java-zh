# 联系我们

我们始终欢迎读者的反馈。

**一般反馈**：如果你对本书的任何方面有疑问，请在邮件主题中注明书名，并发送电子邮件至 `customercare@packtpub.com`。

**勘误**：尽管我们已尽一切努力确保内容的准确性，但错误仍可能发生。如果你在本书中发现错误，我们将不胜感激，请向我们报告。请访问 [www.packtpub.com/submit/errata](http://www.packtpub.com/submit/errata)，选择你的书籍，点击“勘误提交表”链接，并输入详细信息。

**盗版**：如果你在互联网上发现任何形式的我们作品的非法副本，我们将不胜感激，请向我们提供位置地址或网站名称。请通过 `copyright@packt.com` 联系我们，并附上材料的链接。

**如果你有兴趣成为作者**：如果你有擅长的主题，并且有兴趣撰写或为本书做出贡献，请访问 [authors.packtpub.com](http://authors.packtpub.com/)。
