# 总结

本章引导我们了解了一个示例服务网格，它由一个 Web 应用程序、两个在新型 Quarkus 实现中使用 MP 功能的微服务镜像、一个外部 Web 服务以及两个基于 Docker 的服务组成。这展示了您将在云环境中遇到的各种 MP 功能与外部服务之间的相互作用，以及与本地网格环境外部的 Web 服务的集成。这应该能让您体会到使用 MicroProfile API 和实现来组合微服务所涉及的步骤。

在下一章中，我们将了解正在开发中的 MicroProfile 规范，以了解正在探索的方向。虽然这些规范目前不属于 MicroProfile 核心功能集，但它们是未来纳入的候选对象，研究它们将让我们了解 MicroProfile 可能的发展方向。
