# 微服务与云原生与容器原生

首先，让我们从应用开发的角度来区分这三个术语之间的差异。正如我们在第 1 章《Eclipse MicroProfile 简介》中讨论的那样，企业级 Java 微服务具有以下特征：

*   它是使用 Java 语言编写的微服务。
*   它可以使用任何 Java 框架。
*   它可以使用任何 Java API。

*   它必须达到企业级标准，这意味着它必须具有高可靠性、可用性、可扩展性、安全性、健壮性和性能。
*   它必须满足微服务的所有特征，这些特征列于 [`martinfowler.com/microservices/`](https://martinfowler.com/microservices/)。

根据其定义，微服务并未规定底层技术的具体细节……
