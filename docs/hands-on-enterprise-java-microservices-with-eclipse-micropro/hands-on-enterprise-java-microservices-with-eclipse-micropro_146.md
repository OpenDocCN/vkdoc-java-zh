# 其他实现 MicroProfile 的项目

SmallRye 是一个开源项目，它开发了任何供应商或项目都可以使用的 Eclipse MicroProfile 实现。这是一项社区工作，欢迎所有人参与并为 SmallRye 做出贡献，[`smallrye.io`](https://smallrye.io)。例如，社区最近将 Extensions for MicroProfile 项目贡献给了 SmallRye，从而通过为 Config Sources、OpenAPI、Health、JAX-RS 和 REST Client 提供扩展来丰富其功能。

Extensions for MicroProfile 项目网站是 [`www.microprofile-ext.org`](https://www.microprofile-ext.org)，其 GitHub 仓库是 [`github.com/microprofile-extensions`](https://github.com/microprofile-extensions)。

SmallRye 的实现已经过 Eclipse MicroProfile TCK 的测试并已通过。

使用 SmallRye 的开源项目包括 Thorntail ([`thorntail.io`](https://thorntail.io))、WildFly ([`wildfly.org`](https://wildfly.org)) 和 Quarkus ([`quarkus.io`](https://quarkus.io))。
