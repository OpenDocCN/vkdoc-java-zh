# 使用 Eclipse MicroProfile 进行云原生应用开发

什么是云原生应用？通常，**云原生**的定义包含以下特征：

*   设计为松散耦合的服务，例如微服务
*   通过语言无关的通信协议进行交互的松散耦合服务，这使得微服务可以用不同的编程语言和框架来实现
*   轻量级容器，可按需或根据资源利用率指标进行弹性伸缩
*   通过敏捷 DevOps 流程进行管理，云原生应用的每个微服务都经历独立的生命周期，该生命周期通过使用**持续集成/持续交付**（**CI/CD**）管道的敏捷流程进行管理
