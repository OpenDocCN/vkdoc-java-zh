# MicroProfile REST 客户端及其功能简介

**MicroProfile REST 客户端**（**MP-RC**）提供了一种用于对 REST 端点进行类型安全调用的 API。应用程序可以使用它来对其他服务执行远程调用。

它利用 Java 接口上的 JAX-RS 注解来描述与远程服务的实际契约。然后，这些接口用于创建客户端代理，这些代理隐藏了底层 HTTP 通信的许多细节。

MP-RC 规范定义了在 Java 接口上利用 JAX-RS 注解的要求，以及用于增强行为的 MP-RC 特定注解，包括如何传播传入请求头、如何使用提供程序增强 JAX-RS 行为、异常映射、CDI 支持以及与其他 MicroProfile 规范的集成。我们将从定义类型安全端点接口开始，更详细地了解 MP-RC。
