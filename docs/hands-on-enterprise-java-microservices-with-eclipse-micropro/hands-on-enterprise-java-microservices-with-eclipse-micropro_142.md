# KumuluzEE

Sunesis 是开源项目 KumuluzEE 的赞助商，该项目实现了 Eclipse MicroProfile 规范。KumuluzEE 将自己定义为一个使用 Java 和 Java EE 技术的轻量级微服务框架，并且是符合 Eclipse MicroProfile 规范的实现。KumuluzEE 允许你仅使用所需的组件来引导一个 Java EE 应用程序，并且它还支持将微服务打包并运行为 uberjar。就像其他支持 uberjar 的实现一样，你可以通过输入以下命令来运行你的微服务：

```
$ java -jar <可执行 JAR 文件>
```

KumuluzEE 还提供了一个 POM 生成器，它可以创建一个 `pom.xml` 文件，其中包含你计划开发的微服务所需选择的选项和功能。POM 生成器提供了一个清晰有序的列表，列出了 KumuluzEE 支持的配置文件、组件和项目，你可以选择将其包含在 `pom.xml` 文件中。

KumuluzEE 为不同的 MicroProfile API 提供了少量示例。

关于 KumuluzEE 实现 Eclipse MicroProfile 的文档，请参考 [`ee.kumuluz.com/microprofile`](https://ee.kumuluz.com/microprofile)。

最后，KumuluzEE 在 [`ee.kumuluz.com/tutorials/`](https://ee.kumuluz.com/tutorials/) 提供了一些有趣的教程。
