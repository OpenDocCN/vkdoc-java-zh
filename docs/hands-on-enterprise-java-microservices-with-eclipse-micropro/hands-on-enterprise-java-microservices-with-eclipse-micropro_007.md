# 本书涵盖内容

第 1 章，*Eclipse MicroProfile 简介*，在数字经济背景下展开讨论，阐述了什么是企业级 Java 微服务，以及本书后续章节将涵盖的内容。

第 2 章，*治理与贡献*，涵盖了治理、流程以及如何为 MicroProfile 项目做出贡献。

第 3 章，*MicroProfile 配置与容错*，详细介绍了 MicroProfile 子项目中的配置和容错，解释了每个子项目解决的问题，并给出了相应的代码示例。

第 4 章，*MicroProfile 健康检查与 JWT 传播*，将带你了解健康检查和 JWT 传播子项目，借助代码示例帮助你理解它们所应对的挑战。

第 5 章，*MicroProfile 指标与 OpenTracing*，讨论了指标和 OpenTracing 子项目及其解决的问题。你将学习如何使用这些子项目的代码示例。

第 6 章，*MicroProfile OpenAPI 与类型安全 REST 客户端*，涵盖了另外两个子项目：OpenAPI 和类型安全 REST 客户端。本章将帮助你使用代码示例，并理解这些子项目在解决企业问题中的实用性。

第 7 章，*MicroProfile 实现、Quarkus 以及通过会议应用实现的互操作性*，详细介绍了当前市场上可用的 MicroProfile 实现，并讨论了该项目迄今为止的进展。它还深入探讨了*会议应用*，这是一个展示不同供应商 MicroProfile 实现集成的演示程序。

第 8 章，*一个可运行的 Eclipse MicroProfile 代码示例*，提供了一个使用 MicroProfile（基于*会议应用*）开发的完整可运行项目，并提供了下载所述资源的坐标。

第 9 章，*响应式编程与未来发展*，回顾了当前正在孵化/开发的 API，以及正在讨论未来纳入 MicroProfile 规范的 API。此外，它还涵盖了用于响应式编程的 MicroProfile 候选 API，以及 MicroProfile 与 Jakarta EE 之间潜在的未来关系。

第 10 章，*多云环境中的 MicroProfile*，讨论了 MicroProfile 如何成为云中基于微服务的应用的优秀规范，并为在混合云和多云环境中使用 MicroProfile 提供了指导和建议。
