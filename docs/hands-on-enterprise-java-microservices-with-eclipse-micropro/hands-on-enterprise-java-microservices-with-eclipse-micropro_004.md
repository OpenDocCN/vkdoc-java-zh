# 关于作者

**Cesar Saavedra** 自 1990 年起从事 IT 行业，拥有计算机科学硕士学位和 MBA 学位。在他的职业生涯中，他曾担任开发人员、顾问、技术销售和技术营销人员。他目前负责 Red Hat JBoss EAP、Eclipse MicroProfile、OpenJDK、Quarkus 和 Jakarta EE 的技术产品营销。他还负责运行时、集成、**业务流程管理**（**BPM**）和规则管理组合的技术营销，并与工程和产品管理部门在思想领导力方面密切合作。Cesar 撰写了白皮书、电子书和博客文章，并担任会议和网络研讨会演讲者，向客户和合作伙伴进行演示。
