# Hammock

John Ament 是开源项目 Hammock 的赞助商，该项目实现了 Eclipse MicroProfile 规范。与 Thorntail 类似，Hammock 是一个生成 uberjar 的应用程序组装器。要运行 uberjar，你需要输入以下命令：

```
$ java -jar <可执行 JAR 文件>
```
