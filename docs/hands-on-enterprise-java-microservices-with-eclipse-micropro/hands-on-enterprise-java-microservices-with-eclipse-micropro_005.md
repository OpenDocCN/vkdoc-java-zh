# 关于审校者

**David R. Heffelfinger** 是一位独立顾问，常驻华盛顿特区。他是 Java 冠军、NetBeans 梦之队成员，并且是 JavaOne 内容委员会的成员。

他撰写了多本关于 Java EE、应用服务器、NetBeans 和 JasperReports 的书籍。他之前的著作包括《使用 NetBeans 8 进行 Java EE 7 开发》、《使用 GlassFish 4 应用服务器的 Java EE 7》和《面向 Java 开发者的 JasperReports 3.5》等。David 曾多次在 JavaOne 和 Oracle Code 等会议上发表演讲。

他还曾在旧金山和蒙特利尔的 NetBeans Day 上发表演讲，展示增强 Java EE 应用程序开发的 NetBeans 功能。你可以在 Twitter 上关注他：@ensode。

**Yogesh Prajapati** 是一位工程师，在可扩展和分布式企业应用程序的架构、设计和开发方面拥有丰富经验。

他撰写了由 Packt 出版的《Java Hibernate Cookbook》一书。他在 Java、Spring 和云开发（如 REST 和微服务）的不同方面拥有超过 8 年的经验，并熟练掌握包括 Backbase、Hibernate、AWS Cloud、Google Cloud、Node.js、JavaScript、Angular、MongoDB 和 Docker 在内的技术/框架。

他目前在 dotin Inc.（The Personality DNA Company）担任首席全栈顾问。

他毕业于古吉拉特大学，获得计算机应用硕士学位。你可以在他的博客 kode12 上关注 Yogesh。
