# MicroProfile 健康检查和 JWT 传播

在本章中，我们将介绍 MicroProfile 健康检查和 **JSON Web 令牌**（**JWT**）传播项目。健康检查项目关注于将应用程序定义的运行状况暴露给外部世界，而 JWT 传播则关注于定义一种可互操作的安全令牌以及在应用程序中使用该令牌。在本章中，你将了解这些规范所解决的问题、它们的结构以及如何在你的应用程序中使用它们。本章中的代码片段仅供参考。如果你想要该规范的工作代码版本，请参阅第 8 章，*一个可运行的 Eclipse MicroProfile 代码示例*。

我们将涵盖以下主题：

*   什么是健康检查
*   MicroProfile 健康检查如何暴露健康检查端点以及对该端点的查询格式
*   如何为你的应用程序编写 MicroProfile 健康检查
*   MicroProfile JWT 传播中令牌所需的格式
*   我们如何利用 MicroProfile JWT 传播进行安全决策
