# 使用 Jaeger 进行追踪

到目前为止，我们只讨论了检测的不同方面。然而，要运行完整的追踪基础设施，我们需要一个追踪后端。在本节中，我们将使用 Jaeger（[`www.jaegertracing.io/`](https://www.jaegertracing.io/)）来演示收集到的追踪数据如何在追踪系统中呈现。我们选择 Jaeger 是因为 Thorntail 提供了与 Jaeger 的直接集成。其他供应商可以提供与其他系统的集成，例如 Zipkin 和 Instana。几乎每个追踪系统都提供追踪的甘特图样式视图（或时间线）。对于追踪新手来说，这个视图可能令人不知所措，但它是在分布式系统中系统分析调用的绝佳工具。
