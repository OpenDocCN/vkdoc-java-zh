# Quarkus

开源项目 Quarkus 于 2019 年首次亮相。Quarkus 是一个 Kubernetes 原生 Java 栈，可以编译为原生机器语言或构建到 HotSpot (OpenJDK)。使用 Quarkus 时，你的应用程序消耗的内存非常少，性能出色，能够处理高吞吐量的调用，并且启动时间非常快（即，启动加上首次响应时间），这使得 Quarkus 成为容器以及云原生和无服务器部署的绝佳运行时。Quarkus 还提供了一个扩展框架，允许对库和项目进行 *quarking*，使它们能够与 Quarkus 无缝协作。

Quarkus 的使命是将你的整个应用程序及其使用的库转换为最优化的...
