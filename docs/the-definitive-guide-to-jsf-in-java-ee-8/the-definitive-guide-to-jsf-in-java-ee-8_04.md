# 关于作者

![A454457_1_En_BookFrontmatter_Figb_HTML.jpg](img/A454457_1_En_BookFrontmatter_Figb_HTML.jpg)

**鲍克·肖尔茨**（Bauke Scholtz）是 Oracle Java 冠军、JSF 2.3 专家组成员，以及 JSF 辅助库 OmniFaces 的主要创建者。在互联网上，他更广为人知的名字是 BalusC，是 Stack Overflow 上的顶级用户和贡献者之一。鲍克已将多项 OmniFaces 解决方案集成到 JSF 2.3 中。他是一名 Web 应用专家，曾为或正在为 Mercury1 Limited、MyTutor、Nava Finance、LinkPizza、ZEEF、M4N/Zanox、ITCA、RDC 等来自金融科技、联盟营销、社交媒体等领域的客户提供咨询服务，拥有 17 年的从业经验。本书为鲍克提供了一个深入解答使用 JSF 时最常见问题，并正确解决最常见难题的机会。

![A454457_1_En_BookFrontmatter_Figc_HTML.jpg](img/A454457_1_En_BookFrontmatter_Figc_HTML.jpg)

**阿尔扬·泰姆斯**（Arjan Tijms）就职于 Payara Services Ltd，是 JSF（JSR 372）和安全 API（JSR 375）专家组成员。他是广受欢迎的 JSF 库 OmniFaces 的联合创建者（该库曾获 2015 年 Duke 选择奖），并且是一套被多家 Java EE 厂商使用的 Java EE 认证 SPI（JASPIC）测试套件的主要创建者。阿尔扬拥有荷兰莱顿大学的计算机科学硕士学位。撰写本书主题对他而言是自然而然的选择；他已在博客上撰写了许多相关内容，并希望通过参与本书的编写来进一步拓展这些内容。
