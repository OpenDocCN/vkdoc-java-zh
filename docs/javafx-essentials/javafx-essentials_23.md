# 总结

到目前为止，你已经了解了 JavaFX 是什么，并看到了它的强大之处。你已经成功下载并安装了 Java 8 JDK 和 NetBeans IDE。在成功安装必备软件后，你通过 NetBeans IDE 创建了一个 JavaFX Hello World GUI 应用程序。在学习了如何编译和运行 JavaFX 应用程序之后，你对源文件 `HelloJavaFX.java` 进行了快速的代码走查。

接下来，在第 2 章 *JavaFX 8 要点与创建自定义 UI* 中，你将了解 JavaFX 8 的架构组件和引擎，它们使 JavaFX 应用程序能够在底层高效、流畅地运行。你还将学习最常见的布局 UI 组件，并了解如何为整个应用程序或单个场景节点设置主题。

我们还将介绍 Java SE 8 最重要的特性——Lambda 表达式及其工作原理。然后，我们将深入了解 **Scene Builder**，将其作为声明式 UI 和高效工具，接着学习生成的基于 FXML 的标记文档，以及如何将其导入 NetBeans IDE，以继续实现你的应用程序逻辑，并将其与你已在 FXML 文档中声明的 UI 控件关联起来。

最后，你将能够创建一个未随默认 JavaFX 8 UI 控件一起提供的自定义 UI 组件。
