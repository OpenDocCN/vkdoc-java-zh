# 关于作者

**Mohamed Taman**，e-finance 公司的首席架构师和软件开发经理，居住在埃及开罗。他毕业于开罗大学工程学院电气工程专业。他是一位经验丰富的 Java 开发者，曾从事 Web、移动和物联网领域的开发，涉及金融、银行、旅游、政府和医疗保健等行业。在此之前，他曾在辉瑞、Intercom Enterprise（IBM 金牌合作伙伴）、Silicon Expert 和 Oracle 工作，使用过多种技术，包括面向用户的 GUI 前端、后端、中间件以及大型系统的集成。

他热衷于在许多会议上演讲，推广 Java 标准并分享他的全球经验，他是 Java 社区的活跃成员，自 2015 年起成为 Java 冠军。

此外，Mohamed 还是 Adopts Java EE 8、OpenJDK 和 JavaFX 项目的成员。他曾是 Java 社区流程组织的执行成员，是首位加入其董事会的非洲人。

他还是 JSR 354、363 和 373 专家组的成员。他也是 EGJUG 和 MoroccoJUG 的领导者，以及 Oracle 埃及架构师俱乐部的董事会成员。他获得了 2014 年 Duke 选择奖和第 11 届年度 JCP adopt 2013 奖项。
