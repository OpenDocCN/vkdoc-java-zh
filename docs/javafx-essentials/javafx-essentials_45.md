# 为什么树莓派是 JavaFX 的完美选择？

那么，是什么让树莓派成为 Java 和 JavaFX 的完美选择呢？答案可以从以下几点中找到：

*   它的运行速度比规格参数所显示的要快。如前所述，树莓派的默认时钟频率为 900 MHz。但凭借其可安全超频至 1 GHz 的 900 MHz 时钟频率和 1 GB 内存，这台小型计算机能够运行真正庞大且强大的应用程序。
*   非常明智的是，树莓派基金会选择了一款支持浮点运算的 CPU（通常称为硬浮点），其性能优于所谓的纯软浮点芯片。适用于树莓派的操作系统充分利用了这一额外的性能和速度优势。
*   最后，树莓派拥有一个相当强大的图形处理单元（GPU），其快速的 3D 核心能够以 40MBits/s 的 H.264 编码实现蓝光画质播放（[`www.raspberrypi.org/help/faqs/#generalSoCUsed`](https://www.raspberrypi.org/help/faqs/#generalSoCUsed)）。
