# 关于审校者

**Sergey Grinev** 是一位经验丰富的软件开发和 QA 工程师，专注于为 Java 平台构建可靠的质量流程。他在 Oracle 工作期间开始涉足该领域，负责 JavaFX 的质量。过去几年，他一直在 Azul Systems 工作，负责其定制 JVM 的质量。

此外，Sergey 喜欢教学，在各种 Java 会议上演讲，授课，并在 [`stackoverflow.com`](http://stackoverflow.com) 上回答与 Java 相关的问题。

他毕业于圣彼得堡国立大学，目前居住在俄罗斯圣彼得堡。

**José Pereda** 拥有结构工程博士学位，并在西班牙巴利亚多利德大学工业工程学院担任助理教授。他的热情在于应用编程解决实际问题。自 1999 年开始使用 Java，他现在是 JavaFX 的倡导者，开发商业应用和开源项目（JFXtras、FXyz、[`github.com/jperedadnr`](https://github.com/jperedadnr)），合著了一本 JavaFX 书籍（*JavaFX 8 Introduction by Example*），撰写博客（[`jperedadnr.blogspot.com.es/`](http://jperedadnr.blogspot.com.es/)），发推文（`@JPeredaDnr`），并在会议（JavaOne、JAX、Jfokus、JavaLand 等）上发表演讲。José 与妻子和孩子居住在西班牙巴利亚多利德。
