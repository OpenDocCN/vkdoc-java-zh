# 总结

本章让你深入了解了移动行业，以及如何利用不同的项目（例如用于 **iOS** 的 **RoboVM** 和用于 **Android** 的 **JavaFXPorts**）来开发和定制基于 JavaFX 的应用程序，从而使其能够在两个平台上运行。

接着，我们学习了如何安装 Android 开发所需的软件和工具，以及如何启用 Android SDK 和 *JavaFXPorts* 库，以便将我们基于 JavaFX 的拨号器应用打包并安装到真实的 Android 设备上，并提交到 Google Play 商店。

我们看到了如何使用 CSS 来自定义应用，为同一个应用提供不同的外观和风格，以确保你为 Android 版本提供了不同的 CSS。

接下来，我们学习了如何将设备置于调试模式，以便通过命令行从 `jfxmobile-plugin` 成功安装应用。最后，我们介绍了测试的四条黄金法则。

下一章与本章内容不会有太大差异，但它将为你提供很好的入门指导，让你了解如何将 JavaFX 8 应用定位到 iOS 设备上运行。你还将学习如何使用其开发工具。
