# 第 1 章 JavaFX 8 入门

JavaFX 是 Java 的下一代**图形用户界面（GUI）**工具包。它是一个能够轻松快速构建高性能 Java 客户端应用程序的平台。

JavaFX 的底层引擎通过硬件加速图形利用现代 GPU，同时提供设计良好的编程接口，使开发者能够结合图形、动画和 UI 控件。

这些功能使您能够为客户提供引人入胜、复杂且完全可定制的客户端 GUI，给他们留下深刻印象。

虽然 Java 最初的目标是*嵌入式*和*客户端*领域，但自 2006 年以来，许多原因促使 Java 语言成为企业级领域的顶级开发平台。

但最近，随着 JavaFX 平台作为标准客户端 GUI 的引入，这些最初的目标再次开始流行。

尽管 JavaFX 远不止是一个 GUI 工具包，但它允许 Java 开发者创建具有引人入胜用户界面的客户端应用程序，并轻松连接到后端系统。

此外，JavaFX 灵活的 FXML 支持使您能够轻松构建**MVC**（**模型-视图-控制器**）架构模式的应用程序，并使用 Scene Builder 工具实现所见即所得的方法。

JavaFX 的绑定功能简化了实体间的通信，并进一步支持 MVC。除此之外，JavaFX 还提供了使用 CSS 进行快速、可定制的 UI 建模。

通过添加一个具有文档模型的完整`WebView`组件，映射到 Java 代码变得容易，并为 3D 和媒体功能提供了强大支持。
