# 总结

本章让我们很好地理解了如何利用 RoboVM 为 **iOS** 开发和定制基于 JavaFX 的应用程序，从而使其能够在 Apple 平台上运行。

你了解了 RoboVM 的功能和局限性，以及它的工作原理；同时，你也掌握了可用于开发的技能。

接着，你学习了如何安装 iOS 开发所需的软件和工具，以及如何启用 Xcode 并结合 *RoboVM* 编译器，在操作系统模拟器上打包和安装基于 Phone Dial JavaFX 8 的应用程序。

我们看到了如何重用已在第 4 章《为 Android 开发 JavaFX 应用程序》中开发的同一应用程序，这证明了 Java 的“一次编写，到处运行”范式是可行的。

最后，我们提供了关于如何在真实设备上运行和部署应用程序的提示。

下一章将打开物联网开发世界的一扇窗；我们将了解如何购买 Raspberry Pi 2 型号，安装并配置 *raspbian-wheezy* 操作系统用于开发，以及如何为嵌入式设备安装 Java SE。然后，我们将开发一个 JavaFX 8 应用程序，运行在这张信用卡大小的微型计算机上。
