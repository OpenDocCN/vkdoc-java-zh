# 总结

在本章中，你了解了结合 Arduino 与 JavaFX 所能实现的功能。你首先学习了 Arduino 的基础知识，包括其不同型号、主要规格、购买渠道以及组件构成。随后，我们探讨了更多可供获取项目灵感的网站。

接着，你了解了如何下载并设置 Arduino IDE 以与 Arduino 进行通信。在学习了如何在 IDE 中加载示例程序后，你亲自动手尝试，使用 Arduino Uno 和温度传感器搭建了一个简单电路，创建了草图，并在 IDE 的串口监视器中读取了温度数值。

然后，你学习了如何使用 Java Simple Serial Connector 库从计算机读取串口数据。

你学会了如何使用 JavaFX 折线图 API 来监控和显示 Arduino 的读数。接着，你看到了一个示例，该示例使用 Arduino 板通过温度传感器测量血液温度水平，并将来自串口的一系列数据绘制成 JavaFX 图表。

在下一章中，你将学习如何与计算机进行非接触式交互——无需键盘、鼠标甚至触摸设备等任何输入设备，仅通过手势来控制你的 JavaFX 应用程序。
