# 总结

在本章中，我们拿了一台只有扑克牌盒大小的信用卡尺寸微型计算机，并将其改造成了一台 JavaFX 开关控制器。在此过程中，你了解了树莓派，学会了如何创建可启动 SD 卡、如何为其安装操作系统、如何配置它以获得最佳性能和远程连接能力、如何为其分配固定网络（IP）地址，以及如何从你的开发机连接到树莓派。

我们还重新回顾了如何将 JDK 8/JavaFX 8 安装到树莓派和你的开发机上，并且你学会了如何在开发机上安装和配置 NetBeans，使其能够将树莓派上的 JDK 用作远程调试和执行平台。

在开发机和树莓派都准备就绪后，我们讨论了开发一个简单但出色的应用程序所需的原则，该应用程序使用 JavaFX 和一些精选工具/API（包括 Pi4j）从树莓派控制外部世界，并控制 GPIO。

最后，你看到了如何通过几次点击，从 NetBeans 将应用程序远程部署到你的树莓派上。
