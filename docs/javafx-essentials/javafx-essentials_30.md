# 总结

在本章中，我们涵盖了大量内容——首先简要介绍了 JavaFX 8 架构组件，这些组件是流畅高效运行 JavaFX 应用程序的基础层。接着，我们探讨了如何在场景上渲染图形，并简要说明了最常用的布局和 UI 控件。

您了解了 Java SE 8 中的新特性，如 lambda 表达式和函数式接口，并通过示例展示了各自的强大功能。

您学习了如何使用 `setUserAgentStylesheet(String URL)` 和 `getStylesheets().add(String URL)` 方法，通过自定义 CSS 文件为应用程序设置样式。接下来，您简要了解了 Scene Builder 以及如何将 FXML 加载到场景中。最后，您学习了 JavaFX 中的自定义 UI 组件及其创建方法。

在下一章中，您将学习如何创建包含多个场景的桌面应用程序，以及如何对其进行打包。此外，我们还将学习如何与 Web 交互，并使用 JavaFX 8 开发 Web 应用程序。
