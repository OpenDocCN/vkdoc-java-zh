# 第 3 章 开发 JavaFX 桌面与 Web 应用程序

本章将介绍如何开发引人入胜的桌面和 Web 应用程序，利用多核硬件加速 GPU，交付具有惊艳外观和体验的高性能 UI 应用。

由于 JavaFX 完全用 Java 从头编写，我们将使用一些 Java SE 8 内置核心库来驱动应用程序。此外，我们还将学习如何将应用程序打包为独立应用，以便启动和分发。

同时，我们还将介绍 JavaFX 8 中任何 Web 应用所必需的核心 Web API，例如`javafx.scene.web.WebEngine`、`java.net.HttpURLConnection`和`javafx.scene.web.WebView`。

我们将讨论 JavaFX 与 HTML5 之间的关系，这一点很重要，因为 JavaFX 的 API 与 HTML5 的特性相辅相成。HTML5 是一个富 Web 内容平台，用于创建类似**RIA**（**富互联网应用**）Web 应用的用户体验，同时具备原生桌面软件的特性。

更重要的是，我们将逐步开发桌面版的*笔记应用*，然后将其在 Web 上运行。

此外，我们还将涵盖将*笔记应用作为 Web 应用*部署到桌面和 Web 所需的所有知识和技能。

以下是本章将学到的技能：

*   开发和运行桌面与 Web 应用程序
*   控制应用程序 UI
*   如何打包 JavaFX 8 桌面应用程序
*   在 JavaFX 应用程序中加载 HTML5 内容
*   从 JavaFX 向 JavaScript 发送数据，反之亦然
*   部署 JavaFX Web 应用程序
