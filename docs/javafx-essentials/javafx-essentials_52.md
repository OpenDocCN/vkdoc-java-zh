# 第 7 章 使用 JavaFX 监控和控制 Arduino

**Arduino** 是一个基于简单可编程微控制器板的开源电子工具，可以使用免费的开源 IDE 进行编程。它可以独立运行，也可以连接到计算机，通过从各种开关或传感器获取输入来感知环境，并通过控制各种灯、电机和其他物理输出设备来采取行动。

作为最早的**物联网**（**IoT**）设备之一，它诞生于 2005 年。从物联网概念诞生之初，它就已经存在了。

Arduino 可以独立运行，也可以与计算机上运行的软件（Java、JavaFX、Python 等）进行通信，并且这些板子可以手工组装，也可以购买预组装好的。

事实上，Arduino 简化了使用微控制器的过程。同时，对于教师、学生和感兴趣的业余爱好者来说，它比其他系统更具吸引力，因为它*价格低廉*——一块 Arduino 板的价格不到 50 美元。

一个简单、清晰且易于使用的编程环境；开源且可扩展的软件；以及开源且可扩展的硬件——这些特性，以及其他特点，使得 Arduino 支持“自己动手”和“与他人一起动手”的理念，这正是创客运动的定义。

本章将向你展示如何使用 JavaFX 结合 Arduino 板开发一个桌面应用程序，以监控来自真实世界温度传感器的数据，并将其以图表形式呈现，*你的血液到底有多热！*

在本章中，你将：

*   熟悉 Arduino 板及其组件
*   安装并准备 Arduino 软件、环境、IDE 和驱动程序
*   开发一个 Arduino 血液计量草图，以控制和监控电路
*   使用串行通信将 Arduino 数据读取到 JavaFX 应用程序中
*   使用 JavaFX 图表 API 呈现数据
