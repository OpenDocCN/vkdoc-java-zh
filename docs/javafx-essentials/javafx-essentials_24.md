# 第 2 章 JavaFX 8 核心知识与创建自定义 UI

了解 JavaFX 的核心知识将帮助你轻松构建复杂且精巧的 UI 解决方案。

在本章中，你将简要了解 JavaFX 8 的架构，从而理解 JavaFX 架构组件和引擎如何与你的 JavaFX 应用程序高效协同工作，并流畅地渲染其图形。

你将学习如何在 JavaFX 场景上渲染图形，为此，我们将使用场景、一些控件和样式创建一个基础应用程序。

我们将探讨 Java SE 8 特性的基础知识（例如 **Lambda 表达式**和**函数式接口**），以帮助提高代码的可读性、质量和生产力。

当我们创建了第一个结构化的 JavaFX 8 应用程序后，如果能够在不改变其功能的情况下更改应用程序的 UI，那岂不是很好？你将通过了解 JavaFX CSS 样式的基础知识来学习主题化。

最后，你将了解如何使用 Scene Builder 以图形方式创建和定义 UI 屏幕，并将其保存为 JavaFX FXML 格式的文件。你还将获得创建*自定义控件*的实践经验。

在本章中，我们将涵盖以下主题：

*   理解 JavaFX 架构组件
*   使用 JavaFX 组件设置 UI
*   使用 Java SE 8、Lambda 表达式和其他特性
*   为你的应用程序设置主题以适配不同平台
*   使用 CSS 自定义应用程序 UI
*   使用 Scene Builder 工具以可视化方式创建 UI
*   使用 FXML 构建自定义 UI
