# 前言

本书，正如其标题（*JavaFX 8 精髓*）所暗示的，是一本实用的书籍，为您提供一套强大的基本技能，引导您充满信心地快速构建高性能的 JavaFX 8 客户端应用程序。这些应用程序通过硬件加速图形利用现代 GPU，同时为您的客户提供引人入胜、复杂且精美的富客户端 GUI，这将给他们留下深刻印象。

学习 JavaFX 8 精髓是深入创建应用程序的第一步，这些应用程序最重要的是可以在任何平台上运行，从*桌面、Web、移动设备、平板电脑*到*嵌入式*设备，如*Arduino*、*Raspberry Pi* 和多核开发。遵循 Java 的*一次编写，随处运行*范式，JavaFX 也保留了这一点。因为 JavaFX 8 完全是用 Java 语言从头编写的，您会感到得心应手。

大多数章节都是快节奏的指南，将帮助您在 Java GUI 编程中抢占先机，利用 JavaFX 8 并在任何平台上部署和运行。

在阅读本书示例时，您会发现代码是使用 Java 8（是的，Java SE 8）上的 JavaFX 8 编写的，这样新的 API 和语言增强将帮助您成为更高效的开发者。话虽如此，探索所有新的 Java 8 功能将会非常方便（我鼓励您这样做）。

最后但同样重要的是，您将能够使用 JavaFX 开发令人惊叹的非接触式交互式运动应用程序，这些应用程序可与 Leap motion 设备进行交互。
