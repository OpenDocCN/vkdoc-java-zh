# 总结

到目前为止，我们一直在学习如何为桌面端和 Web 端开发 JavaFX 企业级应用。

在本章中，我们掌握了开发任何应用程序的技能：首先在纸上勾勒布局草图，然后将其转化为实际的、可交互的彩色 UI 原型。我们学习了如何嵌套容器和控件以实现所需的布局。一旦最终开发方案获得批准，我们便通过响应用户操作并实现功能需求，让应用程序真正“活”了起来。

借助 Java SE 8 的强大特性以及 JavaFX 绑定，我们让代码变得更加强大、整洁且简洁。最后，我们学习了如何将应用程序部署到目标桌面用户或 Web 用户，以适应不同的平台和环境。

在下一章中，我们将学习如何为基于 Android 的智能手机开发 JavaFX 应用程序。此外，我们还将掌握下载和安装 Android SDK 工具、与日志记录器、模拟器以及其他工具交互所需的技能，这些技能将有助于你未来进行任何与 JavaFX 无关的移动开发。
