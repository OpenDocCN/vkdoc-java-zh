# 本书读者对象

如果您是一名 Java 开发者，或是一位经验丰富的 Java Swing、Flash/Flex、SWT 或 Web 开发者，希望将您的客户端应用程序提升到新的水平，那么本书适合您。本书将引导您走上正确的道路，开始创建精美、可定制且引人入胜的用户界面。

此外，您还将学习如何快速创建高性能的富客户端应用程序，这些应用程序可以在任何平台上运行，无论是桌面、Web、移动设备还是嵌入式系统，例如树莓派、Arduino，以及基于非接触式 Leap Motion 的应用。

本书是一本快节奏的指南，将帮助您在利用 JavaFX 8 进行 Java GUI 编程方面抢占先机，该技术可部署并在任何平台上运行。
