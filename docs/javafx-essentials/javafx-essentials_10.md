# 本书内容概览

第 1 章 *JavaFX 8 入门*，是对 JavaFX 8 的介绍。本章将 JavaFX 8 作为一种技术进行探讨，阐述为何值得关注它，并介绍其历史、核心特性以及应用场景。

现在是时候准备好合适的工具，并按照必要步骤安装 JavaFX 8 及其配套开发工具了。在本章中，您将了解一些能提高读者生产力的附加工具。作为验证我们方向正确的最后一步，我们将以一个简单的“Hello JavaFX”应用程序来结束本章。

第 2 章 *JavaFX 8 要点与创建自定义 UI*，讨论了没有什么比收到复杂建议作为问题解决方案更令人沮丧的了。正因如此，我一直坚持专注于核心要点。为了在 JavaFX 场景中渲染图形，您需要一个基础应用程序、场景、画布、形状、文本、控件和颜色。

此外，您还将学习作为未来任何应用程序支柱的 JavaFX 8 基本应用结构。最后，我们还将探索一些 Java SE 8 特性（例如 Lambda、Streams、JavaFX 属性等），以帮助提高代码的可读性、质量和生产力。

在获得创建结构化 JavaFX 8 应用程序的实践经验后，如果您能在不改变功能的情况下更改应用程序的 UI，那岂不是很好？在本章中，您将了解主题化，以及如何通过应用各种主题（外观和感觉）来自定义应用程序，并学习 JavaFX CSS 样式的基础知识。

您将使用 Scene Builder 来图形化地创建和定义 UI 屏幕，并将其保存为 JavaFX FXML 格式的文件。最后，您将学习如何创建自定义控件。

第 3 章 *开发 JavaFX 桌面与 Web 应用程序*，涵盖了如何开发一款引人入胜的桌面和 Web 应用程序，该应用程序利用多核硬件加速 GPU，以出色的外观提供高性能的基于 UI 的应用程序。

由于 JavaFX 完全是用 Java 从头编写的，因此我们将使用一些 Java SE 8 内置的核心库来驱动我们的应用程序。此外，您还将学习如何将应用程序打包为独立的应用程序，以便启动和分发。

另外，我们还将介绍任何由 JavaFX 8 驱动的 Web 应用程序中必备的核心 Web API，例如`javafx.scene.web.WebEngine`和`javafx.scene.web.WebView`。

我们还将讨论 JavaFX 与 HTML5 之间的关系，这一点很重要，因为它们相辅相成。JavaFX 丰富的客户端 API，结合 HTML5 丰富的 Web 内容，创造了一种兼具原生桌面软件特性的 RIA Web 应用用户体验。

第 4 章 *开发适用于 Android 的 JavaFX 应用程序*，随着非 PC 客户端的兴起，手机和平板电脑的市场份额正在增长。JavaFX 8 可以为 Web 和桌面提供丰富的客户端应用程序。如果您编写一个 JavaFX 应用程序，请确保您希望它能在尽可能多的设备上运行。本章将为您提供关于 SDK 的必要实践经验和知识，这些 SDK 允许用户为 Android 手机创建原生应用程序。

第 5 章 *开发适用于 iOS 的 JavaFX 应用程序*，是前一章的延伸。如果您为 Android 编写了一个 JavaFX 应用程序，请确保您希望它能在尽可能多的 iOS 设备上运行。本章将为您提供关于 SDK 的必要实践经验和知识，这些 SDK 允许他们为 Apple iOS 创建原生应用程序。

第 6 章 *在树莓派上运行 JavaFX 应用程序*，将为您提供开发能在信用卡大小的计算机——树莓派板上运行的 JavaFX 8 应用程序所需的所有必要技能和知识。随着物联网（IoT）近来成为热门话题，Java 可以说是为物联网而生的。

第 7 章 *使用 JavaFX 监控与控制 Arduino*，涵盖了另一种万物互联（IoT）。Arduino 是一个开源电子原型平台，提供低成本的原型平台，以支持 DIY 理念和创客运动。

本章将为您提供所有必要的技能和知识，以便快速将 JavaFX 与 Arduino 板结合使用，开发用于监控来自现实世界的数据或控制真实设备的桌面应用程序。

第 8 章 *使用 JavaFX 开发交互式 Leap Motion 应用*，将使您了解手势识别。您将发现一个很酷的小工具——Leap Motion 设备，它允许采用非接触式方法来开发增强型 JavaFX 应用程序。

机器用户输入界面正日益摆脱以鼠标为中心的模式，转而支持多点触控甚至非接触式输入。手势是当今人类与机器自然交流的方式之一。

附录 *成为 JavaFX 专家*，将为您提供许多有用的链接和参考资料，帮助您进一步了解关于 JavaFX 的一切。

在本章末尾，请务必查看当今在生产环境中使用 JavaFX 的众多框架、库和项目。
