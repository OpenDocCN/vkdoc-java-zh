# JavaFX 目标

JavaFX 的诞生主要目标是在多种设备上使用，例如嵌入式设备、智能手机、电视、平板电脑和台式机。JavaFX 也遵循 Java 的*一次编写，随处运行*范式。

JavaFX 8 完全用 Java 语言从头编写，让你感觉宾至如归。因此，用 JavaFX 编写的应用程序可以部署在台式机、笔记本电脑、Web、嵌入式系统、手机和平板电脑上。

Oracle 已不再支持嵌入式系统；这项工作留给了 ARM 等公司。从 JavaFX 2.x 到 8.x，移动设备从未得到官方支持；现在之所以存在支持，完全是因为 **OpenJFX**。开源社区将 JavaFX 带到移动环境，使大家从中受益。

有关 OpenJFX 的更多信息，请访问 [`wiki.openjdk.java.net/display/OpenJFX/Main`](https://wiki.openjdk.java.net/display/OpenJFX/Main)。

JavaFX 是一套图形和媒体包，使开发人员能够设计、创建、测试、调试和部署富客户端应用程序，这些应用程序能在不同平台上一致运行，且只需一个捆绑包，无需使用许多单独的库、框架和 API 来实现相同目标。这些单独的库包括媒体、UI 控件、`WebView`、3D 和 2D API。

因此，如果你是一名 Java 前端开发人员，或是一位经验丰富的 Java Swing、Flash/Flex、SWT 或 Web 开发人员，希望将你的客户端应用程序提升到新水平，并想为客户开发一个美观且复杂的用户界面，那么你正在学习 JavaFX 技能的正确道路上——这本书就是为你准备的。
