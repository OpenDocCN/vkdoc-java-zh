# 1. 引言

在本书《通过游戏开发学习 Java》中，你将通过动手实践游戏开发任务来学习 Java 编程语言的基础知识。通过完成针对文本中三款游戏特定方面的主题编程挑战，你将获得使用 Java 编程语言、NetBeans IDE、一个 2D 游戏引擎以及三款不同 2D 游戏的实践经验与知识。闲话少叙，请允许我提供一些关于本书的详细信息，首先从本书的结构和约定开始。

## 关于本书

本书将引导你学习 Java 编程语言的基础知识及部分高级主题。随着我们深入不同主题，你将面临挑战，需要修改、修复、编写和/或调试本书附带的三款游戏之一的副本：

*   Pong 克隆版

*   记忆配对

*   地牢陷阱

每款游戏均使用本书附带的一个专有 2D 游戏引擎编写。该引擎是开源的，如果你想查看相关代码，可以在此处访问：

`github.com/apress/intro-java-through-game-dev`

与此游戏项目相关的源代码可在同一仓库中找到。

每个项目都包含一个功能完整的游戏版本，附带完整源代码以及编程挑战。随着你阅读本书的深入，你将获得一些挑战，这些挑战要求你将特定的 Java 编程语言知识应用于当前游戏项目。每个章节的挑战都预先配置了特定的游戏项目副本。

一般来说，本书不要求你长时间连接互联网，除了下载本书及其相关代码和程序之外。你只需要少量的计算资源即可运行相关软件和游戏。你的计算机应至少满足以下最低配置要求：

*   双核 CPU

*   4GB 系统内存

*   2GB 存储空间

任何相对现代的计算机都能轻松应对这些工作负载。在使用任何现代编程语言时，都必须决定如何用该语言进行编码。通常，你可以仅使用记事本和几个命令行工具来编写和构建程序。然而，在我们的案例中，我们希望使用更高级的工具，因此我们需要使用 IDE。

IDE 是集成开发环境，它是一个相当复杂的程序，其任务是简化软件开发人员的某些工作，使他们能够承担更大的项目并专注于手头的编码工作。游戏是复杂的程序，通常包含许多动态部件。因此，使用 IDE 将使我们受益匪浅，所以我们选择了 NetBeans IDE。

本书中的每个示例游戏都是为一个简单的 Java 2D 游戏引擎编写的。通过完成本书中的编程挑战所获得的经验，将极大地帮助读者打下坚实基础，从而能够使用附带的游戏引擎在 Java 中创建自己的 2D 游戏。
