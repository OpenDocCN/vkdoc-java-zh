# 2. NetBeans 入门

当您想要快速开发应用程序时，可以使用 *Apache NetBeans IDE* 来简化您的任务。在本章中，您将了解编辑器的布局以及一些技巧，这些技巧将帮助您更好地控制代码并加快开发速度。

## 默认布局和用途

IDE 中的每个窗口都以其所在窗格或位置中的选项卡形式出现。此外，IDE 的窗口系统允许您通过拖放操作将窗口排列到图 2-1 布局中所示的任何窗口位置。

### 资源管理器窗口

所有提供对用户对象访问的窗口都使用资源管理器窗口位置。这些窗口包括项目窗口、文件窗口、服务窗口和任务窗口。

*项目窗口*（**Ctrl+1** 或 **Cmd+1**）显示项目源文件。它显示一个或多个项目的逻辑视图。每个项目或视图由一个节点表示，并根据项目类型具有特定的图标（例如，Java 应用程序节点的显示方式与 JavaFX 应用程序节点不同），如图 2-2 所示。

*文档窗口*通常显示在*编辑器窗口位置*。通过文档窗口，你可以编辑 Java 程序文件和其他源文件。你还可以从*Java 平台 Shell* 执行命令，因为 Java 9 Shell 的命令行界面也已集成到文档窗口中。

#### 源代码编辑器

*源代码编辑器*是 IDE 布局中使用最频繁的部分。它是你编写源代码或为 Java 及其他类型应用程序设计图形用户界面的地方。

## 加速您的应用程序编码

Apache NetBeans IDE 拥有大量丰富的功能，可帮助您加快编码任务。IDE 具有默认功能，例如代码补全、语法高亮、代码模板等，您可以通过更改各种设置来使用和修改这些功能。

### 智能代码补全

*代码补全*是 IDE 中最令人兴奋且最常用的功能。它特定于源代码编辑器，并且适用于大多数受支持的语言（Java、HTML、PHP 等）。

有两种方法可以打开代码补全弹出对话框（如图 2-24 所示）：按 `Ctrl+Space` 或在编辑器内键入点（`.`）字符。

![../images/479166_1_En_2_Chapter/479166_1_En_2_Fig24_HTML.jpg](img/479166_1_En_2_Fig24_HTML.jpg)

图 2-24

Java 编辑器中的代码补全

代码补全功能比以往任何时候都更智能。例如，如果您想调用一个变量、类或方法，自动弹出的代码补全对话框将列出所有已定义的元素，并通过键入元素名称的第一个字符帮助您轻松选择所需的元素。

还有*驼峰式补全*。您无需键入所有字符，只需键入您感兴趣的单词的大写字母。然后按 `Ctrl+Space`，驼峰式补全功能将补全元素的完整名称。

您还可以访问元素的 Javadoc。将光标放在元素名称上，然后按 `Ctrl+Space`，如图 2-25 所示。您将看到一个对话框打开，其中包含您所选元素的 Javadoc。

### 重构

有时您想更改项目中某个特定元素（类、变量、方法）的名称，但这可能会产生连锁反应，因为它可能会破坏依赖关系，即依赖于该名称的其他元素。与其修改每个使用了该名称的文件，不如使用*重构*功能。重构一个元素会将所有出现该旧名称的地方替换为新名称。

例如，假设您想将一个类从 *Animals* 重命名为 *Animal*。以下是让 IDE 为您完成此操作的步骤：

![../images/479166_1_En_2_Chapter/479166_1_En_2_Fig28_HTML.jpg](img/479166_1_En_2_Fig28_HTML.jpg)

1.  将光标放在源编辑器中的类名上，然后按 `Ctrl-R`，或者右键单击并从弹出菜单中选择 **重构 ➤ 重命名**（参见图 2-28）。

2.  如果您希望修改也应用于注释中的出现位置，请勾选**在注释上应用重命名**。

![../images/479166_1_En_2_Chapter/479166_1_En_2_Fig29_HTML.jpg](img/479166_1_En_2_Fig29_HTML.jpg)
