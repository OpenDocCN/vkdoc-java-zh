Scala 之旅

欢迎阅读《Java 开发者的 Scala 实用入门指南》。本书将帮助你从 Java 编程过渡到 Scala 编程。本书旨在帮助 Java 开发者入门 Scala，而无需完全掌握所有更高级的函数式编程范式。

Scala 既是一门面向对象语言，也是一门函数式语言。虽然我会讨论函数式编程的一些优势，但本书更侧重于使用命令式 Scala 高效编程，而非深入掌握函数式编程。如果你已经熟悉 Scala，但希望转向纯函数式编程，那么本书可能不适合你。建议阅读 Paul Chiusano 和 Rúnar Bjarnason 合著的优秀著作《Scala 函数式编程》¹。

本书经常在 Java 和 Scala 之间进行“同类对比”。因此，如果你熟悉用某种特定方式在 Java 中完成某项任务，我会展示如何在 Scala 中实现同样的功能。在此过程中，我将介绍 Scala 语言的语法。
