# 第三部分
高级主题

高级主题

本书的最后一部分包含一些额外的可选功能，可以添加到之前的许多项目中，例如游戏手柄控制器输入和粒子系统（一种用于创建独特、非重复动画效果的高效方法）制作的特效。本部分还包含一些涉及高级算法和 3D 图形的游戏项目。

第 13 章：用户输入的替代来源

本章将探讨用户输入的两种替代来源：游戏手柄控制器和触摸屏控制。具体来说，你将把这些替代输入源添加到前几章中介绍的《海星收集者》游戏中。

第 14 章：迷宫游戏

在本章中，你将学习如何创建基于迷宫的《迷宫奔跑者》游戏，其灵感来源于《吃豆人》等街机游戏以及早期主机游戏《迷宫狂热》。本章的主要新概念是用于生成和解决迷宫的算法。

第 15 章：高级 2D 图形

在本章中，你将学习两种将复杂图形融入项目中的技术。第一种技术是粒子系统，它可以创建爆炸等特效；这将被整合到《太空岩石》游戏中，以取代基于精灵表的动画。第二种技术是着色器编程，它通过操作渲染图像的像素来创建模糊或发光等效果；这将被整合到《海星收集者》游戏中。

第 16 章：3D 图形与游戏入门

本章将介绍 LibGDX 的一些 3D 图形功能，以及描述和渲染三维场景所需的概念和类。为了简化和精简这一过程，你将改编一些旧类并编写一些新类来完成所涉及的各种任务。最后，你将创建《海星收集者 3D》游戏，这是本书开头介绍的《海星收集者》游戏的三维版本。

第 17 章：旅程继续

这最后一章将介绍在继续游戏开发过程中需要考虑的各种步骤。其中，你将探索开展其他项目、学习相关领域的技能，以及将你的游戏带给更广泛的受众。在此过程中，本章将提供各类资源列表，以及针对多种情况的通用建议。
