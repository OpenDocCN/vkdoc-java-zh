# 第一部分
基本概念

基本概念

在接下来的章节中，你将了解 LibGDX 库，并在此库的基础上构建一个自定义框架，以简化各类游戏的创建。这些章节是本书其余部分的基础；后续章节将假定你已经掌握了此处所介绍主题的实用知识。

第 1 章：Java 与 LibGDX 入门

本章将解释如何设置 BlueJ，这是一个因其简洁性和用户友好性而被选中的 Java 开发环境。将解释标准的第一个程序（即在文本控制台打印“Hello, World!”）。接着，将给出设置 LibGDX 软件库的说明，并展示一个可视化的“Hello, World!”程序（该程序在一个窗口中显示一幅世界图像）。最后，将详细解释使用 LibGDX 进行游戏开发的一些优势。

第 2 章：LibGDX 框架

本章将首先讨论视频游戏程序的整体结构，包括游戏程序所经历的阶段以及每个阶段必须完成的任务。在创建一个名为“海星收集者”的基础游戏的过程中，将介绍 LibGDX 库的许多主要特性和类。这个游戏是贯穿全书的一个反复出现的示例；在介绍新主题时，会向该项目添加新功能。

第 3 章：扩展框架

在本章中，你将从一个代表游戏实体的核心 LibGDX 类开始，并创建该类的扩展，以支持动画、基于物理的运动、改进的碰撞检测，以及使用列表管理实体多个实例的能力。

第 4 章：射击游戏

本章将展示你所创建框架的强大功能，通过使用该框架制作一个全新的游戏：太空岩石，这是一款受经典街机游戏《小行星》启发的太空主题射击游戏。在此过程中，你将向框架添加功能，加入处理离散动作（如发射激光）输入的能力（与连续动作（如移动）相对）。

第 5 章：文本与用户界面

在本章中，你将学习如何显示文本、创建显示图像或文本的按钮，以及使用表格设计用户界面。首先，你将通过向第 3 章的“海星收集者”游戏添加这些功能来学习这些技能。然后，你将在学习如何创建过场动画（有时称为游戏内动画）的过程中巩固并加强这些技能，这些过场动画为你的游戏提供了叙事元素。在可选的最后一节中，你将创建一个名为“失踪的作业”的视觉小说风格游戏，该游戏侧重于一个故事，并允许玩家决定故事如何发展。

第 6 章：音频

在本章中，你将学习如何向游戏添加音频元素——音效和背景音乐。首先，你将通过向“海星收集者”添加这些功能来了解这些主题。然后，在可选部分中，你将通过一个名为“节奏敲击者”的音乐节奏游戏来巩固这些技能，在该游戏中，玩家按下视觉上指示的一系列按键，这些按键与背景播放的音乐同步。
