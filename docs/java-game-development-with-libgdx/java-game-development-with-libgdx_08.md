# 第二部分
中级示例

中级示例

凭借你在第一部分中打下的坚实基础——对 LibGDX 基本概念和类的掌握，以及你开发并完善的自定义框架——现在你已经准备好创建各种不同类型的视频游戏了，每款游戏都拥有独特的机制。本部分各章节相互独立，可以按任意顺序阅读，但第 11 章和第 12 章除外，因为它们依赖于第 10 章中介绍的瓦片地图。

第 7 章：横版卷轴游戏

在本章中，你将创建一款名为《飞机躲避者》的横版卷轴动作游戏，其灵感来源于《Flappy Bird》和《Jetpack Joyride》等现代智能手机游戏。在此过程中，你将学习如何创建无限滚动的背景效果，利用加速度设置模拟重力，并实现一个随时间推移而增加玩家挑战难度的难度曲线。

第 8 章：弹跳与碰撞游戏

在本章中，你将创建一款名为《矩形毁灭者》的弹球打砖块游戏，其灵感来源于《打砖块》和《Arkanoid》等街机及早期主机游戏。本游戏将实现的新功能包括：使用鼠标移动物体、模拟物体间的弹跳碰撞，以及创建玩家可拾取的道具。

第 9 章：拖放游戏

在本章中，你将学习如何为游戏添加拖放功能；随后你将创建一个包含相关代码的新类。为了展示这个新类的灵活性，你将创建两款利用它的新游戏。第一款是拼图游戏，它由一张被分割成若干碎片的图片组成，玩家需要将这些碎片在网格上正确排列。第二款是一款名为《52 张牌收集》的纸牌接龙游戏，玩家需要将一副标准扑克牌整理成若干堆。

第 10 章：瓦片地图

本章将解释如何使用 Tiled——一款通用地图编辑软件，可用于关卡设计流程的多个方面。然后，你将创建一个类，用于将瓦片地图文件中的数据导入到你已开发的自定义框架中。这些知识将用于改进之前的两个游戏项目：对于《海星收集者》游戏，你将设计一个迷宫般的关卡（使用岩石作为墙壁）并添加一些场景；而对于《矩形毁灭者》游戏，你将设计一个色彩缤纷的砖块布局。本章学到的内容是后续两章的基础。

第 11 章：平台游戏

在本章中，你将学习如何创建平台游戏《跳跃杰克》，其灵感来源于《大金刚》和《超级马里奥兄弟》等街机及主机游戏。本章引入的新概念包括：具有多种动画的游戏实体、平台物理、使用额外演员作为“传感器”来监测物体周围区域的重叠与碰撞、可穿越平台，以及钥匙与锁机制。

第 12 章：冒险游戏

本章包含了整本书中最具雄心的游戏项目：一款名为《寻宝探险》的基于战斗的冒险游戏，其灵感来源于《塞尔达传说》等经典主机游戏。这款游戏使用了新功能，例如：使用两种不同武器（剑和弓箭）的敌人战斗、根据游戏状态（例如剩余敌人数量）显示不同信息的非玩家角色（NPC），以及一个道具商店机制。
