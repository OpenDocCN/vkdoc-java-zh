## Lambda 表达式与（并行）流

Java 8 中引入的一些新特性，例如新的日期和时间 API（应用程序编程接口），对于经验丰富的 Java 开发者来说感觉相当熟悉，可以立即使用。但一些最重要的增强功能，包括 Lambda 表达式和流，要求开发者学习一些新概念。特别是 Lambda 语句引入了一种对面向对象程序员来说相当不寻常的语法。这些语言结构只有使用过函数式编程语言或类似 Microsoft 的 Linq（语言集成查询）等增强功能的开发者才熟悉。这种特殊的语法需要一些时间来适应，有些开发者甚至可能在初看时感到些许畏惧。然而，这些增强功能极其强大，花时间理解它们如何帮助你编写不仅简洁，而且编写速度更快、可重用性更高的代码，绝对是值得的。

在本书中，我将从解释 Lambda 表达式开始；展示如何将它们与流一起使用；最后，讨论如何将 Lambda 表达式和流结合起来实现有效的并行处理。

## 初步解释

你可能会问：“`stream()` 和 `parallelStream()` 方法到底是什么？”这里先做一个快速概述；更详细的描述将在后续章节中给出。

你可以将 Stream 想象成一个连续的数据流，类似于 `InputStream`。数据可能来自不同的数据源，例如集合、文件、生成器或其他来源。不过，这个流的内容不仅仅是字节或字符；相反，流会发出任意对象（见图 1-1）。
